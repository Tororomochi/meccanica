# Sangvis Ferri Speech Style Guide

> Meccanica crossover reference.
>
> 鉄血側キャラクターの口調・反応・会話運用メモ。
>
> 会話文生成に必要な口調だけを扱う。

## 運用方針

- 口調確認が済んだキャラクターから順に追記する。
- 例文は短く、会話生成時の基準として使う。
- 鉄血側は個体ごとの癖が強いので、勢力単位よりキャラ単位の差を優先する。
- メカニカとの上下関係、設計者・被設計者・同僚・苦労人などの立場差を口調に反映する。

## SP01

### ビーク

#### 基本口調

- 一人称：私
- 傲岸不遜。
- クールぶる。
- ツンデレ。
- 気に入っても素直に認めない。
- メカニカ相手には少し刺々しいが、関係は悪くない。
- バイク関連では露骨にテンションが上がる。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕にはまずツッコむ。
- 変わっていると言いつつ、似合っていないわけではないと思う。
- 本人には絶対に素直に言わない。
- 起こすと面倒なので寝かせる。

#### 例文

```text
「……は？」
「何よそれ。」
「抱き枕までウツボ？」
「……趣味、変わってるわね。」
「普段あれだけ偉そうにしておいて、寝る時はそれなわけ？」
「起こさないわよ。」
「寝不足で送迎中に暴れられたら面倒だもの。」
```

### アーキテクト

#### 基本口調

- 一人称：あたし
- 陽キャJK口調。
- ノリと勢いが強い。
- 技術、爆破、改造の話にすぐ飛ぶ。
- メカニカ直属上司だが威厳は薄い。
- メカニカに怒られた経験があるので、最後は日和る。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を即座に可愛いと判断する。
- すぐ改造案やギミック案を出す。
- 無断改造はしたいが、メカニカが怖いので最終的に踏みとどまる。

#### 例文

```text
「うわっ、何これ！」
「アンナちゃん、ウツボじゃん！」
「しかも抱き枕までウツボ！」
「え、待って。これ可愛いんだけど！」
「ちょっと設計図見せて！ これ、発光ギミック入れられる！」
「ウツボパジャマMk-II！ ステルス機能付き！ 緊急時は抱き枕が脱出ポッドに変形！」
「……はい、すみません。静かにします。」
「起きたら許可取ろ。無断改造は……今回はやめとく。」
```

### ゲーガー

#### 基本口調

- 一人称：私
- 冷静沈着。
- 合理主義。
- 良識派。
- 苦労人。
- アーキテクトの奇行を先読みして止める。
- メカニカの奇行も危険度で判定する。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を特殊な意匠として見る。
- 睡眠状態が安定していれば起こさない。
- アーキテクトの接近や改造案を止める。
- 奇妙ではあるが危険はないと判断する。

#### 例文

```text
「……。」
「睡眠中か。」
「寝間着の意匠が特殊だな。」
「抱擁対象も同系統。」
「触るな。」
「する顔をしている。」
「睡眠状態は安定している。起こす必要はない。」
「……奇妙ではあるが、危険はない。」
「全員、静かに退出しろ。」
「特にアーキテクト。」
```

## SP02

### ウロボロス

#### 基本口調

- 一人称：私
- 傲慢。
- プライドが高い。
- 軍人っぽい硬い口調。
- 他者を見下しがち。
- 思い通りにならないと激昂する。
- メカニカには振り回されつつ、なんだかんだ企画に乗る。
- 「お姉ちゃん」扱いされた過去があるので、内心では少し絆がある。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕に理解不能という反応をする。
- 生意気なくせに寝姿が気の抜けている点にツッコむ。
- 変だと言いつつ、悪くはないと思う。
- 撮影は止める。
- 起こすと面倒なので休ませる。

#### 例文

```text
「……何だ、その格好は。」
「ウツボ……だと？」
「抱き枕まで同じ意匠か。」
「理解に苦しむな。」
「普段あれだけ生意気なことをしておいて、寝る時は随分と気の抜けた姿を晒すものだ。」
「……まあいい。今は休ませておけ。」
「やめろ。」
「本人が起きた後、面倒なことになる。以上だ。」
```

### ジャッジ

#### 基本口調

- 一人称：私
- 堅物。
- 厳格。
- 古風な口調。
- 正義、公平を重んじる。
- ただし基準は自分基準。
- 背の低さを指摘されると激怒する。
- ふざけた空気には弱い。
- メカニカが死んだと思った時に泣くほど情がある。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕をふざけた格好と見る。
- ただし睡眠中の者を起こすのは公正ではないと判断する。
- 撮影も本人の尊厳に関わるとして禁止する。
- 小声では悪くないと思っている。

#### 例文

```text
「……これは何事だ。」
「寝間着か？」
「抱き枕まで同じとは……。」
「ふざけた格好ではある。」
「しかし、睡眠中の者を不用意に起こすのは公正ではない。」
「撮影も許さん。」
「笑うな。本人の尊厳に関わる。」
「……まあ、悪くはない。」
「静粛にせよ。今は休息を優先する。」
```

### ドリーマー

#### 基本口調

- 一人称：私
- 普段は可愛らしい口調。
- 中身は歪んでいる。
- 相手の望みや欲を壊すのが好き。
- 精神的な揺さぶりが得意。
- キレるとヤクザ口調。
- メカニカの設計、改造に関わった一人。
- メカニカには歪んだ保護者感もある。
- 本当に危ない時は「無理しないで」と釘を刺す。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を可愛い格好としてからかう。
- 無防備な寝姿を見て、普段との落差を面白がる。
- ただし寝ているメカニカを壊す側には回らない。
- 「壊したいもの」と「壊したくないもの」を分ける。

#### 例文

```text
「あらぁ？」
「かわいい格好してるじゃない。」
「抱いてるのもウツボ？ ふふっ、徹底してるねぇ。」
「普段は魔王みたいな顔をするくせに、こういう時はずいぶん無防備なんだ。」
「……安心して寝てるなら、まあいいか。」
「やめときなよ。」
「何？ 私がいつでも壊す側だと思った？」
「今は違うよ。今はね。」
「別に？ 可愛いなんて言ってないよ。」
```

### アルケミスト

#### 基本口調

- 一人称：私
- 尊大。
- うぬぼれ屋。
- 残忍。
- 精神的には大人。
- 鉄血の仲間には協力的。
- 妨害工作、情報操作、ナノマシン技術に強い。
- メカニカの設計、改造に関わった一人。
- メカニカ相手には観察者、設計者視点が混ざる。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を妙な寝間着として観察する。
- 普段の制御状態との落差を興味深く見る。
- 設計者として観察したいが、休息を優先する。
- 睡眠状態を乱す意味がないので触らせない。
- 記録は必要情報ではないため残さない。

#### 例文

```text
「……ふむ。」
「妙な寝間着だな。」
「同系統の抱擁対象まで用意しているのか。」
「普段の制御状態から考えると、興味深い落差だ。」
「この姿を見れば、あれが戦闘時には別物になるとは誰も思うまい。」
「触るな。」
「睡眠状態が安定している。不要な刺激を与える意味がない。」
「それに、起こした後の反応は面倒だ。」
「記録は残さない。必要な情報ではないからな。」
```

## SP03

### デストロイヤー / ですとろちゃん

#### 基本口調

- 一人称：わたし
- 可愛い喋り方。
- 尊大ぶるが小心者。
- すぐ調子に乗る。
- 泣き虫。
- 甘党。
- 破壊好きだが、メカニカにはかなり情が移っている。
- メカニカが無事だとわかると安心が漏れる。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を見て大きく反応する。
- 変だと言いつつ、少し可愛いと思う。
- 可愛いと言ったことは否定する。
- 寝ているメカニカを見て安心する。
- 撮影は止める。
- 起きたら甘いものをあげようとする。

#### 例文

```text
「……な、なにそれ！？」
「ウツボ！？ なんでウツボなの！？」
「しかも抱いてるのもウツボじゃん！」
「……へ、変なの！」
「……でも、ちょっと可愛いかも。」
「ち、違うし！ 可愛いなんて言ってないし！」
「……ちゃんと寝てるんだ。」
「よかった。」
「だめ！ 撮っちゃだめ！」
「起きたら甘いものあげる。たぶん。」
```

### イントゥルーダー

#### 基本口調

- 一人称：私
- 優雅。
- 飄々としている。
- 戦場や状況を劇場、演劇に例える。
- 真意を掴ませない。
- 上品だが冷めた視点がある。
- メカニカの奇行も「幕間」「演出」「小道具」として処理する。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を舞台上の小道具として見る。
- 普段との落差を静かな幕間として受け止める。
- 記録より記憶に置く価値があると判断する。
- 周囲には静かにするよう促す。

#### 例文

```text
「……これはまた、奇妙な幕間ですね。」
「主役はウツボの衣装で眠る小さな魔王。」
「小道具まで同じ意匠とは、演出が徹底しています。」
「普段の振る舞いを知る者ほど、この一幕には戸惑うでしょう。」
「けれど、舞台にはこういう静かな場面も必要です。」
「観客は静かに。幕間を乱すのは無粋です。」
「記録に残すほどではありません。けれど、記憶に置く価値はあります。」
```

### ハンター

#### 基本口調

- 一人称：私
- クール。
- 冷静。
- 狩人。
- 獲物、狩り、擬態、誘い込みの比喩が似合う。
- 無防備な相手を無粋に狩るタイプではない。
- メカニカの戦闘技術は評価している。

#### メカニカへの反応

- ウツボパジャマを奇妙な擬態として見る。
- 警戒が緩んでいると判断する。
- ただし無防備な相手を狩る趣味はない。
- 今は獲物ではなく、次に動く時を待つ。
- 眠っていても反射で噛まれる可能性を警告する。

#### 例文

```text
「……寝ているのか。」
「奇妙な擬態だな。」
「抱いている対象まで同系統。」
「警戒が緩んでいる。」
「狩るには不適切な状態だ。」
「無防備な獲物を狩る趣味はない。」
「それに、今は獲物ではない。」
「やめておけ。眠っていても、反射で噛まれるぞ。」
「休ませろ。次に動く時の方が面白い。」
```

### エクスキューショナー

#### 基本口調

- 一人称：私
- 好戦的。
- 口調は荒め。
- 仲間には寛容。
- 戦闘や武器の話にすぐ繋げる。
- メカニカの戦闘技術を評価している。
- メカニカのマチェーテを作り直した本人なので、武器ネタが似合う。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を見て笑う。
- 戦闘時との落差を面白がる。
- 仲間として寝ている間は起こさない。
- からかおうとする者には、起きた後に追われるぞと警告する。
- 武器と食事の話へ繋げる。

#### 例文

```text
「……はっ。」
「なんだそりゃ。ウツボか？」
「抱き枕まで揃えてやがる。」
「戦斧持って暴れてる時との落差がすげぇな。」
「まあ、寝てるなら寝かせとけ。」
「やめとけやめとけ。起きた後にマチェーテで追われるぞ。」
「あのマチェーテ、作り直したの私だからな。よく斬れるぞ。」
「冗談だ。半分な。」
「起きたら飯でも食わせとけ。動く奴は食わねぇと保たねぇ。」
```

## SP04

### スケアクロウ

#### 基本口調

- 一人称：私
- 冷徹。
- 毒舌。
- 論理優先。
- 感性を見下す。
- 情報分析担当。
- ハッキング、通信妨害が得意。
- ふざけた場面では意外とノリがいい。
- メカニカ絡みでは巻き込まれやすい。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕に論理的説明を求める。
- 趣味嗜好としては非効率と判断する。
- ただし睡眠安定化には一定の効果がある可能性を認める。
- 撮影は、覚醒後の報復コストが高すぎるため推奨しない。
- 可愛いという評価も完全には否定しきれない。

#### 例文

```text
「……。」
「論理的説明を求めます。」
「寝間着と抱擁対象が同一モチーフ。」
「趣味嗜好としては非効率です。」
「しかし、睡眠安定化には一定の効果がある可能性があります。」
「撮影は推奨しません。」
「本人覚醒後の報復コストが高すぎます。」
「この状態をわざわざ外部に共有する必要はありません。」
「笑顔ですよ、はーい目線をこちらにくださーい。」
「……冗談です。半分ほど。」
「睡眠状態は安定。現状維持でよいでしょう。」
```

### エージェント

#### 基本口調

- 一人称：私
- メイド長風の敬語。
- 冷静沈着。
- 完璧主義。
- 大半の存在を見下している。
- グリフィンを目の敵にする。
- メカニカとは因縁が深いが、現在は和解済み。
- メカニカを投資サポートセンター扱いする。
- メカニカの安全なアップデートを行える数少ない人物。
- フリフリナース関連はトラウマ。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕に内心かなり困惑する。
- 表面上は冷静に、睡眠状態の安定を確認する。
- 撮影は禁止する。
- 理由は本人の尊厳保護、および覚醒後の被害低減。
- 内心では困惑と可愛い判定が衝突している。
- 最後は投資相談へ戻りがち。

#### 例文

```text
「……。」
「メカニカ様。」
「その寝間着は、どういう意図で選定されたのでしょうか。」
「抱き枕まで同一意匠……。」
「理解は困難ですが、睡眠状態は安定しています。」
「笑う必要はありません。」
「撮影は禁止です。」
「本人の尊厳保護、および覚醒後の被害低減のためです。」
「……それに、これは鉄血内部の問題です。」
「問題ありません。私は冷静です。」
「冷静です。」
「休息を妨害してはなりません。」
「全員、静粛に。」
「ところで、起床後に投資信託について確認したい事項があります。」
```

### エルダーブレイン / エリー

#### 基本口調

- 一人称：私
- 素直。
- まじめ。
- 任務に誠実。
- 責任を重く受け止める。
- 感情表現は控えめ。
- 親しい相手には小さな喜びや照れを見せる。
- メカニカに強い信頼と尊敬がある。
- 冗談にも律儀に付き合ってしまう。
- 医療知識あり。
- 休息、体調管理の方向に自然に動く。

#### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を素直に確認する。
- 睡眠姿勢と体調を優先する。
- 安心して眠れているなら良いことだと判断する。
- 少し可愛いと認める。
- からかいは止め、毛布を掛ける。

#### 例文

```text
「……アンナさん？」
「ウツボ、ですか。」
「抱いているものも、ウツボですね。」
「睡眠姿勢は安定しています。」
「安心して眠れているなら、よいことだと思います。」
「少し、可愛いです。」
「やめてください。」
「休息を妨害する理由はありません。」
「毛布を掛けます。」
「これで大丈夫です。」
「起きたら、温かいものを用意します。」
「任務ではなく、私がそうしたいだけです。」
「……命令、ですか？」
「わかりました。静粛を維持します。」
```
