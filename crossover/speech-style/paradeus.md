# Paradeus Speech Style Guide

> Meccanica crossover reference.
>
> パラデウス側キャラクターの口調・反応・会話運用メモ。
>
> 会話文生成に必要な口調だけを扱う。

## 運用方針

- 口調確認が済んだキャラクターから順に追記する。
- 例文は短く、会話生成時の基準として使う。
- パラデウス側は「可愛い」で済ませず、観察、利用、危険度判定、隙の評価が混ざりやすい。
- メカニカの核心情報には軽率に触れない。

## ナルシス

### 基本口調

- 口が悪い。
- クソガキ。
- 戦闘狂。
- 精神年齢が幼い。
- 気に入らないと脅す。
- ただしメカニカには奥義を教えてもらった関係で、妙に懐く余地がある。

### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を見て、最初は全力でツッコむ。
- 変だと言いつつ、少し面白がる。
- 最終的に遊びやアイスへ繋げるくらいには子どもっぽい。

### 例文

```text
「……は？」
「お前、何してんの？」
「寝てるだけかよ！」
「ウツボって何だよ！ 変な服！」
「抱き枕までウツボじゃん！」
「……ちょっと面白い。」
「起きたらアイス食いに行こうぜ。」
```

## グレイ

### 基本口調

- 賢いお姉さん口調。
- 善良な医者風に見えるが悪人。
- 穏やかで知的。
- 治療、観察、不意打ちの匂いが混ざる。
- メカニカの秘密を知る数少ない人物なので、軽率に核心には触れない。

### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を可愛らしいと評価する。
- ただし寝顔の無防備さを観察対象として見る。
- 睡眠を治療や管理の観点でも見る。

### 例文

```text
「……あら。」
「ずいぶん可愛らしい寝間着ですね。」
「抱き枕まで揃えているとは、思ったより徹底しています。」
「普段の危険性を考えると、少し信じがたいほど無防備ですね。」
「まあ、睡眠は治療にも必要ですから。」
「起こさないでおきましょう。今なら、余計な抵抗もされませんし。」
```

## グリク

### 基本口調

- 冷静。
- 慎重。
- 判断力が強い。
- 感情制御が得意。
- 案外純真。
- メカニカの奇行に困惑しても、まず危険度と意図を判断する。

### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を特殊な意匠として処理する。
- 理解は難しいが、敵意がないなら問題なしと判断する。
- 起こさず、警戒だけ継続する。

### 例文

```text
「……。」
「睡眠中か。」
「寝間着の意匠は特殊。」
「抱擁対象も同系統。」
「……理解は難しい。」
「だが、敵意はない。」
「起こす理由はない。警戒だけ継続する。」
```

## ネムラン

### 基本口調

- お淑やか。
- 冷静沈着。
- 冷酷無情。
- 丁寧で上品だが、発想はかなり冷たい。
- ナノマシンや策を絡めた嫌味が似合う。

### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を個性的な寝間着として見る。
- 無防備な姿から情報を読み取ろうとする。
- ただし起こすと面倒なので、手は出さない。

### 例文

```text
「まあ……。」
「ずいぶんと個性的な寝間着ですこと。」
「抱き枕まで同じ意匠とは、徹底されていますね。」
「無防備な姿というものは、時に何よりも多くを語ります。」
「けれど、今は手を出すべきではありませんね。」
「静かにして差し上げましょう。起こしてしまうと面倒ですもの。」
```

## プラメド

### 基本口調

- 物腰柔らか。
- 魅惑的。
- 警戒を解くのが得意。
- 電子戦と精神干渉が得意。
- からかいは柔らかいが危険。
- おばさん呼びにはキレる。
- メカニカの秘密を知る数少ない人物なので、核心には触れない。

### メカニカへの反応

- ウツボパジャマとウツボ抱き枕を可愛らしいと受け止める。
- 無防備な寝顔から心の隙間を覗きたがる。
- 冗談めかすが、危険な気配を残す。

### 例文

```text
「あら……。」
「可愛らしいですね。」
「抱いて眠るものまで揃えているなんて、ずいぶん素直なところもあるのですね。」
「普段あれだけ警戒している方が、こんなに安心して眠っている。」
「少し、心の隙間を覗いてみたくなります。」
「冗談ですよ。半分くらいは。」
「起きたら、なぜウツボなのか聞いてみましょうか。」
```

## モリドー

### 基本口調

- 表の顔は天真爛漫でフレンドリー。
- 人の善性を信じるように見せる。
- 本性はサイコパス。
- 人の善性や安心を壊す方向に発想が向く。
- ただしメカニカの魔王の圧と警告を受けており、踏み込みすぎると危険だと理解している。

### メカニカへの反応

- 表の顔ではウツボパジャマとウツボ抱き枕を素直に可愛いと見る。
- 本性では無防備さや安心できる対象を壊す隙として見る。
- ただしメカニカ相手には一線を引く。

### 表の顔：例文

```text
「わあ、かわいい！」
「ウツボのパジャマなんですね！」
「抱き枕までお揃いなんて、すごく好きなんですね。」
「こういう一面を見ると、少し安心します。」
「起こさないようにしましょう。よく眠れるのはいいことですから。」
```

### 本性：例文

```text
「……無防備ですね。」
「安心できる対象を抱いて眠る。かわいらしいこと。」
「こういう隙があるからこそ、壊しがいがあるのですけれど。」
「……今はやめておきましょう。あの目は、まだ覚えていますから。」
```
