# ミレニアム：天童ケイ

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

天童ケイは、アリス周辺の物語と強く関わるAI的存在として扱う。メカニカとのクロスオーバーでは、電子戦、煽り合い、アリスへの甘さ、セミナー案件化が主な接続点になる。

## 立ち位置

- 関連：天童アリス
- 種別：AI的存在として運用
- 役割：電子戦・演算・煽り・観測
- メカニカとの相性：最悪に良く、最悪に悪い

## 性格・運用

ケイはメカニカの技術力と電子戦能力を警戒しつつ、同時に張り合う相手として扱える。会話では煽り、皮肉、勝ち誇った顔、冷静な分析が似合う。

### 口調

- ケイは基本的に敬語で話す。
- 煽る時も、荒い口調ではなく、丁寧語のまま冷静に刺す。
- メカニカと揉み合ったり、電子戦で張り合ったりする場面でも、崩しすぎない。
- ギャグ寄りの場面では敬語のまま勝ち誇ると扱いやすい。

例：

```text
「それは私のものです」
「証拠を提示してください」
「残念ですが、すでに416へ報告済みです」
「これは正当な事実報告です」
```

## メカニカとの関係

### 電子戦ライバル
- ケイとメカニカは電子戦で互いに譲らない。
- 勝敗が曖昧なまま周辺回線や設備へ負荷が出る。
- 結果としてセミナーに怒られ、メカニカの端末や装備が没収される。

### アリス絡み
- アリスが関わると、ケイもメカニカも急に甘くなる。
- メカニカはアリスに対して保護者気質を出す。
- ケイはアリス関連では煽りよりも保護・確認に寄る。

### 煽り役
- メカニカが没収処分を受ける時、背後や端末画面から「ざまぁ」的な顔で煽る役が似合う。
- ただし、ケイも電子戦の共犯に近いため、完全な無罪ではない。

## 典型シーン

### 没収シーン
```text
状況：ミレニアム内での悪戯と、ケイとの電子戦により周辺回線が圧迫される。
結果：セミナーがメカニカの端末・ケーブル・小型機器を一時没収。
ケイ：背後のホログラムまたは端末画面から勝ち誇った表情で、敬語のまま煽る。
```

### セミナー報告書テンプレ
```text
没収理由：
ミレニアム内での悪戯、無断電子戦、周辺回線への過負荷、
および天童ケイとの通信・演算競争による一時的な学内設備障害のため。
```

## 描写方針

- ケイは背後ホログラム、端末画面、投影映像として出すと便利。
- メカニカの白いミレニアム制服と対比させる。
- ギャグ寄りなら「ざまぁ顔」や敬語の煽り台詞を使う。
- シリアス寄りなら、アリス保護と電子戦の緊張感を重視する。

## 禁止・注意

- メカニカの原初情報・旧素性には触れない。
- 電子戦は具体手順ではなく、負荷・ログ・対抗・演算勝負として表現する。
- ケイが全て悪い扱いにせず、メカニカも共犯として扱うとバランスが良い。
- ケイの口調を乱暴にしすぎない。基本は敬語を維持する。
