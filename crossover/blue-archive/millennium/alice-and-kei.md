# ミレニアム：アリス＆ケイ 深掘り

> Blue Archive crossover reference for Meccanica.
>
> `game-development-club.md` と `kei.md` から派生した、アリス＆ケイ周辺の個別運用ファイル。

## 概要

天童アリスと天童ケイは、メカニカのクロスオーバー運用において非常に重要な接続点になる。

アリスは、メカニカの保護者気質・ノーキル方針・ゲーム的比喩・教育者モードを引き出す相手。

ケイは、メカニカの電子戦・煽り合い・AI的対抗心・セミナー案件化を引き出す相手。

## 天童アリス

### 立ち位置
- 所属：ミレニアムサイエンススクール
- 部活：ゲーム開発部
- 役割：勇者的価値観、ゲーム的世界理解、保護対象にもなる中心人物

### メカニカとの関係
- メカニカを「強い味方」「隠しボス」「先輩NPC」のように見る。
- メカニカはアリスに対して非常に甘くなりやすい。
- アリスの前では、メカニカの厳しさよりも保護者気質が出る。
- 戦術や技術の話を、ゲーム的な比喩で説明しやすい。

### 典型シーン
```text
アリス：メカニカ先輩は、隠しボスですか？
メカニカ：俺は味方NPCだぞ。
アリス：では、仲間加入イベントですね！
メカニカ：そういうことでいい。
```

## 天童ケイ

### 立ち位置
- 関連：天童アリス
- 種別：AI的存在として運用
- 役割：演算、電子戦、煽り、観測、アリス保護

### メカニカとの関係
- 電子戦ライバル。
- 互いに煽り合うが、アリスが絡むと急に甘くなる。
- セミナー案件の共犯になりやすい。
- メカニカが没収処分を受ける時、背後ホログラムや端末画面から勝ち誇った表情で煽る役。

### 典型シーン
```text
状況：メカニカの端末類がセミナーに没収される。
ケイ：背後のホログラムで勝ち誇った顔をする。
メカニカ：お前も共犯だろ。
ノア：はい。ログ上はその通りです。
ユウカ：二人とも反省文です。
```

## 三者関係

### アリスがいない場合
- メカニカとケイが電子戦で張り合う。
- 周辺設備に負荷が出る。
- ヴェリタスが観測する。
- セミナーが怒る。

### アリスがいる場合
- メカニカもケイも急に保護者寄りになる。
- 煽り合いは減る。
- アリスのゲーム的理解に合わせて説明する。
- ギャグから急に優しい空気になる。

## メカニカ側の扱い

### 出やすい面
- ノーキル方針
- 保護者気質
- 教育者モード
- 技術をゲーム比喩へ変換する能力
- ケイ相手の負けず嫌い

### 出してはいけない面
- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 原初人物照合

## 没収理由テンプレ

```text
没収理由：
ミレニアム内での悪戯、天童ケイとの電子戦、周辺回線への過負荷、
およびアリスへの説明を口実にした未承認技術実験のため。
```

## 描写方針

- アリス絡みは温かく、ゲーム的な言葉を多めにする。
- ケイ絡みは電子戦・煽り・ログ・没収を使う。
- アリスとケイが同時にいる場合、メカニカはかなり甘くなる。
- セミナーが後処理を担当するとギャグとして安定する。
