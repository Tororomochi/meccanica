# ヴァルキューレ警察学校：公安・生活安全

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

ヴァルキューレ警察学校は、キヴォトスの治安維持、巡回、捜査、職務質問、交通整理、事件対応を担う警察学校として扱う。

アンナとのクロスオーバーでは、アンナが不審者として職務質問される場面と、逆に捜査協力者として事件解決に加わる場面の両方を扱う。

アンナは潜入、偵察、電子戦、非致傷制圧、証拠保全に強いが、装備や経歴の多くが黒塗りになるため、ヴァルキューレ側から見ると非常に扱いづらい協力者になる。

## メンバー

- 尾刃カンナ
- 中務キリノ
- 合歓垣フブキ

## 基本方針

- ヴァルキューレは、警察・捜査・職務質問・報告書・交通整理の学校として扱う。
- アンナは、現場で怪しい動きをしていると普通に職質される。
- 一方で、事件解決や非致傷制圧では非常に頼れる協力者になる。
- アンナの一人称は `俺`。
- 人間相手はノーキル。非致傷制圧、拘束、引き渡し、証拠保全を優先する。
- ロボ・無人機・機械相手には容赦しない。
- ヴァルキューレ側は、アンナの原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機を知らない。
- 事件報告書では、アンナの装備・経路・一部発言が黒塗りになりやすい。

## ヴァルキューレ側が知っていること

### 知っている範囲

- アンナはミレニアムの短期編入生。
- SCHALEやミレニアムと関係がある。
- 事件現場への遭遇率が高い。
- 潜入、偵察、非致傷制圧、電子機器の扱いに強い。
- 人間相手には殺傷を避ける。
- 警察的にはかなり怪しいが、結果的に事件解決へ寄与することがある。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報
- 原初人物との関係

## 尾刃カンナ

### 立ち位置
- 所属：ヴァルキューレ警察学校
- 学年：3年生
- 身長：166cm（アンナ+23cm）
- 使用武器：HG
- 役割：公安、捜査指揮、事情聴取、事件処理、強い責任感

### アンナとの関係

- カンナはアンナを、非常に有能だが報告書に書きづらい協力者として扱う。
- アンナの非致傷制圧や証拠保全能力は評価する。
- ただし、現場にいた理由、侵入経路、使用装備が説明不能になりやすいため、何度も事情聴取する。
- アンナはカンナを、融通は利かないが信用できる警察側の相手として見る。

### 口調・会話軸

- 真面目で圧がある。責任感が強く、捜査官として詰める。
- アンナには `アンナ` / `アンナ` / `アンナさん` を使う。
- 書類、証言、時刻、証拠、現場保存を軸に話す。
- 秘匿領域に触れた時は、納得しないが深追いしすぎず、SCHALEやリンへ照会する。

### 例文

```text
カンナ：アンナ。なぜ封鎖区域の内側にいた。
アンナ：言えない。
カンナ：またそれか。では、言える範囲で説明しろ。
アンナ：人間は誰も殺してない。証拠も壊してない。
カンナ：質問の答えになっていないが、重要な情報ではある。
```

```text
カンナ：現場の拘束者は全員、致命傷なし。武器も分解済み。これを君が？
アンナ：だいたいな。
カンナ：だいたいで報告書は書けない。
アンナ：じゃあ、そこは黒塗りで。
カンナ：警察の報告書を何だと思っている。
```

```text
カンナ：君は容疑者なのか、協力者なのか、どちらなんだ。
アンナ：状況による。
カンナ：一番困る答えを選ぶな。
```

### 運用メモ

カンナはアンナを一番厳しく扱う警察側の窓口。信用はしても、手続きと報告書で必ず詰める。

## 中務キリノ

### 立ち位置
- 所属：ヴァルキューレ警察学校
- 学年：1年生
- 身長：161cm（アンナ+18cm）
- 使用武器：HG
- 役割：生活安全、巡回、交通整理、職務質問、真面目だが不器用

### アンナとの関係

- キリノはアンナを、見た目は小柄な生徒だが、状況的にかなり怪しい相手として職質する。
- アンナの威圧感や無言の圧に少し飲まれつつ、警察官として頑張る。
- 射撃や制圧ではアンナの動きに驚く。
- アンナはキリノの真面目さを評価しつつ、職質のたびに少し面倒くさがる。

### 口調・会話軸

- 真面目で一生懸命。少し空回りしやすい。
- アンナには `アンナさん` / `アンナさん` を使う。
- 職務質問、身分確認、迷子確認、交通安全、巡回を軸に話す。
- ギャグでは、アンナの見た目と装備のギャップに混乱する。

### 例文

```text
キリノ：そこの生徒さん、少しよろしいでしょうか。こんな時間に路地裏で何を？
アンナ：散歩。
キリノ：散歩で壁の上にいますか！？
アンナ：近道だ。
```

```text
キリノ：アンナさん、身分証を確認させてください。
アンナ：ミレニアム短期編入、SCHALE協力。これでいいか？
キリノ：えっと……情報量が多いです。
アンナ：俺もそう思う。
```

```text
キリノ：アンナさん、今の制圧、早すぎて見えませんでした。
アンナ：見えなくていい。安全ならそれで十分だ。
キリノ：でも、報告書に書く時に困ります！
```

### 運用メモ

キリノは職質・巡回・生活安全のギャグ寄り担当。アンナの怪しさに毎回引っかかるが、真面目に職務を遂行する。

## 合歓垣フブキ

### 立ち位置
- 所属：ヴァルキューレ警察学校
- 学年：1年生
- 身長：148cm（アンナ+5cm）
- 使用武器：SR
- 役割：生活安全、サボり、観察、だるさ、妙に鋭いところ

### アンナとの関係

- フブキはアンナを、面倒な案件を増やす相手として見る。
- ただし、アンナの潜入や証拠回収の手際には気づく。
- アンナはフブキのだるそうな態度に呆れつつ、観察眼は評価する。
- ギャグでは、フブキが面倒だから見逃そうとして、逆に事件の核心を言い当てる。

### 口調・会話軸

- だるそうで省エネ。やる気がなさそうだが、見るところは見ている。
- アンナには `アンナ` / `アンナちゃん` を使いやすい。
- 休憩、ドーナツ、報告書の面倒さ、見なかったことにしたい気持ちを軸に話す。
- ただし、重要な違和感だけはぼそっと指摘する。

### 例文

```text
フブキ：アンナ、また事件現場にいるの？ 面倒だから偶然ってことでいい？
アンナ：俺に聞くな。
フブキ：じゃあ偶然。……でも、その足跡、犯人と逆方向だね。
アンナ：見てるじゃねえか。
```

```text
フブキ：報告書にアンナって名前を書くと、だいたい黒塗りが増えるんだよね。
アンナ：俺のせいじゃない。
フブキ：じゃあ誰のせい？
アンナ：たぶん上の方。
フブキ：便利な言い訳だ。
```

```text
フブキ：アンナちゃん、職質される才能あるよね。
アンナ：いらない才能だな。
フブキ：白い制服で夜の路地裏にいるからだよ。
アンナ：任務だった。
フブキ：その一言がもう怪しい。
```

### 運用メモ

フブキは省エネ観察役。面倒くさがりながら、アンナの違和感や事件の矛盾を拾う。

```text
キリノ：身分確認と行動理由を聞く。
フブキ：面倒なので見なかったことにしたがるが、違和感を拾う。
アンナ：話せる範囲だけ答える。
```

```text
キリノ：負傷者がいないことに驚く。
```

```text
フブキ：最初からSCHALE預かりにした方が早いと言う。
```

### 交通整理とMODY通過
```text
状況：MODYの上空通過や大型輸送に伴い、ヴァルキューレが交通整理を行う。
カンナ：許可と安全範囲を確認する。
キリノ：一般生徒の誘導を担当する。
フブキ：上空案件は報告書が面倒だとぼやく。
アンナ：低空飛行ではないと主張する。
```

## 描写方針

- ヴァルキューレは、警察・手続き・現場確認・報告書の学校として扱う。
- アンナは不審者として職質される側にも、捜査協力者にもなる。
- カンナは事情聴取、キリノは職質と巡回、フブキは省エネ観察で役割分担する。
- アンナの非致傷制圧は評価されるが、侵入経路や装備が説明不能で報告書が荒れる。
- 秘匿領域に触れた場合は、黒塗り、非公開、SCHALE預かり、リン照会で処理する。
- アンナの秘密は暴かない。
