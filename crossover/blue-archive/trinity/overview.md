# トリニティ総合学園：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

トリニティ総合学園は、礼節、格式、信仰、秩序、政治、救護、治安維持、放課後の日常が強く混ざる学校として扱う。

百鬼夜行が祭りと怪異、ミレニアムが技術と合理性、ゲヘナが混沌と現場処理に寄るのに対し、トリニティは「きちんとした場」「礼儀」「建前」「審議」「祈り」「規則」が強い。

アンナとのクロスオーバーでは、アンナの白いミレニアム制服、異常な戦闘・潜入能力、軍機境界、悪戯癖、ノーキル方針が、トリニティ側の礼節・規律・祈り・治安維持とぶつかる。

## 基本方針

- アンナのキヴォトス側公開名は `メジロアンナ`。
- 会話では `アンナさん` / `アンナちゃん` / `アンナさん` / `アンナちゃん` を自然に使う。
- アンナの一人称は `俺`。
- トリニティ側はアンナを、ミレニアムから来た問題の多い短期編入生、またはSCHALE経由の協力者として扱う。
- 原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機は出さない。
- 人間相手はノーキル。非致傷制圧、拘束、撤退誘導、先生への引き渡しで扱う。
- ロボ・無人機・機械相手には容赦しない。
- トリニティの生徒は全員女子高校生として扱う。男子生徒は出さない。
- アロナとプラナは先生以外には基本的に認識できない。

## トリニティでのアンナの立ち位置

### 表向き

- ミレニアムの短期編入生。
- 白いミレニアム制服を着た小柄な生徒。
- 技術、電子戦、潜入、非致傷制圧に強い。
- ミレニアムで何度か問題を起こしており、SCHALEやセミナーが対応している。
- トリニティでは、礼儀正しく振る舞える時と、悪戯で台無しにする時の差が激しい。

### トリニティ側が知らないこと

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- アンナの本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報
- 原初人物との関係

### トリニティ側が察してよいこと

- ただの短期編入生ではない。
- ミレニアムとSCHALEが一定の警戒をしている。
- 戦闘・潜入・電子機器に強い。
- 人間相手には殺意を向けない。
- 話せない範囲が極端に多い。
- 礼儀や格式を理解できるが、従うかどうかは別。

## 主な接続軸

### ティーパーティーとの政治・礼儀

ティーパーティーは、アンナを政治的リスク、外交的な客人、ミレニアムからの問題児として扱う。

```text
状況：アンナがトリニティ内で騒動を起こし、ティーパーティーに報告が上がる。
ナギサ：紅茶会の場で遠回しに確認する。
ミカ：アンナの小柄さと強さの差に驚く。
セイア：秘密を暴かず、沈黙の境界を読む。
アンナ：話せる範囲だけを選び、核心は避ける。
```

### 正義実現委員会との治安・制圧

正義実現委員会は、アンナの悪戯・潜入・非致傷制圧を現場で扱う。

```text
状況：アンナがトリニティ内で騒動に巻き込まれる。
ツルギ：直感で危険度を察知する。
ハスミ：規則と許可を確認する。
マシロ：観測と記録を行う。
イチカ：事情聴取と現場処理を担当する。
アンナ：逃げようとして捕まるか、逆に現場鎮圧を手伝う。
```

### 救護騎士団との救護・保護

救護騎士団は、この設定ではアンナのことを深く知らない。負傷者や要保護者として接する。

```text
状況：騒動後、アンナが救護所に連れてこられる。
ミネ：救護対象として保護する。
セリナ：休息と処置を勧める。
ハナエ：明るく検査しようとする。
アンナ：自分は大丈夫だと言うが、座らされる。
```

### シスターフッドとの祈り・沈黙

シスターフッドも、この設定ではアンナのことを深く知らない。秘密を暴くのではなく、話さないまま祈る。

```text
状況：アンナが礼拝堂で静かにしている。
サクラコ：秩序と祈りの場として受け止める。
ヒナタ：道案内や荷物運びで自然に助ける。
マリー：理由を聞かず、静かに寄り添う。
アンナ：何も話せないが、その沈黙に少し救われる。
```

### 放課後スイーツ部との日常・甘味

放課後スイーツ部は、トリニティでアンナを最も日常側に引き戻しやすい。

```text
状況：アンナがタルトゥーフォや副司令スイーツを持ち込む。
アイリ：素直に喜ぶ。
ヨシミ：商品名にツッコむ。
ナツ：甘味を記憶や物語にたとえる。
カズサ：ぶっきらぼうに食べつつ、詮索しない。
アンナ：変な名前でも味を認められて少し得意げになる。
```

### 補習授業部との疑念・補習・孤立

補習授業部は、トリニティの政治的疑念や排除の空気に巻き込まれた生徒たちとして、アンナと接続しやすい。

```text
状況：アンナがSCHALE経由で補習授業部の騒動に同行する。
ヒフミ：アンナを心配しつつ、補習から逃がさない。
アズサ：アンナの実戦的な動きを見て警戒する。
コハル：規則違反や窓侵入に怒る。
ハナコ：アンナの沈黙と話題転換から境界線を読む。
アンナ：疑われる側の空気を嫌い、行動で信用を作る。
```


## 描写方針

- トリニティでは、礼儀と秩序がアンナの雑さや悪戯癖とぶつかる。
- 政治的な話はティーパーティー、現場処理は正義実現委員会、医療は救護騎士団、沈黙はシスターフッド、日常は放課後スイーツ部、疑念と補習は補習授業部へ分ける。
- アンナの秘密は暴かない。
- 話せない範囲に踏み込んだ場合は、発話ロック、話題逸らし、先生への確認、沈黙で処理する。
- トリニティ側は、アンナを危険視しつつも、最終的には生徒として扱う。
- ギャグ時は、商品名、紅茶会、規則違反、補習、ペロロ、礼儀作法で崩す。
- シリアス時は、祈り、沈黙、非致傷制圧、政治的配慮、補習授業部の孤立で締める。
