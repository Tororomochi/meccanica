# 百鬼夜行連合学院：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

百鬼夜行連合学院は、祭り、観光、和風文化、伝承、怪談、ゆるやかな自治を持つ学校として扱う。

ミレニアムやゲヘナのように強い中央集権で動くのではなく、複数の部活・組織がそれぞれの役割を持ち、状況に応じてゆるく連携する。

アンナとのクロスオーバーでは、MODYの百鬼夜行空域通過、祭りへの出店、忍術研究部との潜入技術比較、百花繚乱との調停・非致傷制圧、花鳥風月部を絡めた怪談・認識系シリアスが主な接点になる。

## 基本方針

- 百鬼夜行側は、アンナを「ミレニアムから来た妙に礼儀正しいが怪しい短期編入生」として扱う。
- アンナのキヴォトス側公開名は `メジロアンナ`。
- 会話では `アンナさん` / `アンナちゃん` / `アンナさん` を自然に使う。
- アンナの一人称は `俺`。
- 原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機は出さない。
- 人間相手はノーキル。非致傷制圧、拘束、撤退誘導、後処理で扱う。
- ロボ・無人機・機械相手には容赦しない。
- 百鬼夜行の生徒は全員女子高校生として扱う。男子生徒は出さない。

## 百鬼夜行でのアンナの立ち位置

### 表向き

- ミレニアムから来た短期編入生、またはSCHALE経由の協力者。
- 白いミレニアム制服を着た小柄な生徒。
- 技術力、潜入技術、非致傷制圧能力が異常に高い問題児。
- 祭りや出店では、鉄血チョコ、副司令ブリトー、副司令ロールなどを持ち込む変な子。

### 百鬼夜行側が知らないこと

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- アンナの本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報

### 百鬼夜行側が察してよいこと

- ただの短期編入生ではない。
- ミレニアム、SCHALE、複数学園が一定の警戒をしている。
- 戦闘・潜入・電子機器に強い。
- ただし、子供や一般生徒にはかなり優しい。
- 祭りや食べ物が絡むと、意外と普通に楽しむ。

## 主な接続軸

### MODY航空許可

百鬼夜行空域をMODYが通過する場合、陰陽部や百花繚乱、SCHALE経由で許可・注意喚起を行う。

```text
状況：MODYが百鬼夜行上空を通過する。
陰陽部：空域通過の手続きを確認する。
百花繚乱：安全確保と現場警備を担当する。
お祭り運営委員会：祭りの演出と勘違いしかける。
アンナ：低空飛行しない、祭り会場上空を避ける、などの条件を飲む。
```

### 祭り・屋台

百鬼夜行の祭りに、アンナが鉄血チョコや副司令シリーズを持ち込む。

```text
状況：祭りの出店にアンナが参加する。
シズコ：商機を感じる。
フィーナ：任侠映画っぽい売り文句に乗る。
ウミカ：祭り全体の導線を気にする。
アンナ：変な名前の商品を堂々と並べる。
```

### 忍術・潜入

忍術研究部とアンナの潜入技術が接続する。

```text
状況：イズナがアンナを忍者仲間だと思い込む。
イズナ：純粋に弟子入りまたは勝負を申し込む。
ミチル：動画映えを狙う。
ツクヨ：大きさ・気配・距離感に悩む。
アンナ：忍術ではなく潜入技術だと訂正しつつ、結局付き合う。
```

### 百花繚乱との調停

アンナが百鬼夜行内の騒動に巻き込まれた場合、百花繚乱が調停・確認に入る。

```text
状況：アンナが騒動の現場で非致傷制圧を行う。
百花繚乱：現場確認と事情聴取を行う。
アンナ：やったことは認めるが、軍機に関わる部分は伏せる。
SCHALE：必要に応じて調整に入る。
```

### 怪談・認識系

花鳥風月部や百鬼夜行の怪談と、アンナの認識阻害・魔王の圧・軍機境界が接続する。

```text
状況：百鬼夜行内で、アンナに似た怪談が広がる。
花鳥風月部：噂や怪異として扱う。
アンナ：心当たりがあるが、核心は話せない。
陰陽部：祭りや学園運営への影響を気にする。
```

## 描写方針

- 百鬼夜行は、怪異・祭り・和風文化・観光の空気を優先する。
- アンナは礼儀を守れるが、潜入癖と悪戯癖で騒動を起こす。
- 百鬼夜行側は、アンナの危険性をすぐ敵意に変えず、まず「妙な客人」「祭りの変な協力者」「怪談っぽい子」として受け止める。
- シリアス時は、百花繚乱や花鳥風月部で扱う。
- ギャグ時は、祭り運営委員会や忍術研究部で扱う。
- 軍機領域に踏み込ませない。
