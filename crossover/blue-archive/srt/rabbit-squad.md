# SRT特殊学園：RABBIT小隊

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

RABBIT小隊は、SRT特殊学園の元生徒で構成される小隊として扱う。

SRT全体を広げすぎず、本資料ではRABBIT小隊だけを扱う。アンナとのクロスオーバーでは、野外生活、都市潜伏、偵察、制圧、指揮系統、階級認識、装備運用、即席共同作戦が主な接点になる。

RABBIT小隊は、キヴォトス側では珍しく、アンナの「単なる生徒ではない指揮階級」をある程度知っている例外枠にする。ただし、原初情報・旧素性・年齢・帰還事故・2025年関連情報・本編側軍機までは知らない。

重要：RABBIT小隊がアンナの階級や指揮経験を知っていても、アンナがRABBIT小隊への司令権を保持しているわけではない。RABBIT小隊の指揮権は小隊長であるミヤコ側にあり、アンナは作戦協力者・上位指揮経験者として助言や共同判断を行う立場に留まる。

## 基本方針

- RABBIT小隊は、アンナの階級・指揮経験を一部把握している。
- アンナはRABBIT小隊への司令権を保持していない。
- RABBIT小隊の直接指揮はミヤコが担う。
- アンナの助言や判断は尊重されるが、命令権として扱わない。
- ただし、その情報はSRT式の作戦上必要な範囲に限る。
- アンナを「ミレニアムの短期編入生」としてだけでなく、現場指揮経験を持つ特殊な協力者として扱う。
- アンナの一人称は `俺`。
- 人間相手はノーキル。非致傷制圧、拘束、撤退誘導、情報回収を優先する。
- ロボ・無人機・機械相手には容赦しない。
- RABBIT小隊はアンナの原初情報、旧素性、年齢、帰還事故、2025年関連情報を知らない。

## RABBIT小隊が知っていること

### 知っている範囲

- アンナは単なるミレニアム生徒ではない。
- 非公式ながら、現場指揮・訓練・作戦立案の経験がある。
- グリフィン側で副司令格として扱われることがある。
- 戦術人形の訓練・運用経験がある。
- 潜入、偵察、電子戦、非致傷制圧に強い。
- 通常の生徒扱いではなく、作戦協力者・上位指揮経験者として扱う必要がある。

### 誤解してはいけないこと

- アンナはRABBIT小隊の司令官ではない。
- アンナはRABBIT小隊への恒常的な命令権を持たない。
- `副司令` 呼称はアンナ側の所属・経歴に由来するもので、SRT内階級ではない。
- RABBIT小隊がアンナの判断を採用する場合も、それはミヤコが小隊長として判断した結果である。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機の詳細
- SII、MODY、Scadente Muro の本質的詳細
- 原初人物との関係

## 呼称・階級扱い

- 通常時：`アンナ` / `アンナさん` / `アンナさん`
- 作戦時：`副司令` / `アンナ副司令` / `作戦協力者`
- 公共の場：階級呼称は避け、`アンナさん` または `アンナさん`
- 秘匿場面：必要に応じて `副司令` を使う
- `副司令` と呼ぶ場合でも、RABBIT小隊への司令権を意味しない

```text
ミヤコ：公共の場では、アンナさんと呼びます。
アンナ：その方がいい。副司令は目立つ。
サキ：というか、目立つ以前に説明が面倒になる。
アンナ：そこは否定しない。
```

```text
サキ：念のため確認します。副司令と呼ぶとしても、RABBIT小隊の指揮権はミヤコにありますよね。
アンナ：当然だ。俺はお前らの司令官じゃない。
ミヤコ：はい。助言は受けますが、小隊判断は私が行います。
モエ：つまり、偉いけど命令する人じゃない副司令？
アンナ：雑だが、だいたい合ってる。
```

## 月雪ミヤコ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 学年：1年生
- 身長：156cm（アンナ+13cm）
- 使用武器：RABBIT-31式短機関銃（SMG）
- 役割：小隊長、冷静な指揮、作戦判断、責任感

### アンナとの関係

- ミヤコはアンナを、現場経験を持つ上位指揮経験者として扱う。
- ただし、命令権を渡すのではなく、RABBIT小隊の小隊長として自分の判断を保つ。
- アンナはミヤコの冷静な判断と責任感を評価する。
- 指揮判断が衝突する場合は、作戦目的と民間被害の最小化で擦り合わせる。

### 口調・会話軸

- 冷静で丁寧。感情を抑え、任務・状況・判断を優先する。
- アンナには通常時 `アンナさん`、作戦時 `副司令` を使う。
- 上位指揮経験者として敬意は払うが、SRTとしての独立判断は捨てない。
- 階級や肩書より、現場で何をすべきかを確認する。

### 例文

```text
ミヤコ：副司令、RABBIT小隊は南側から迂回します。正面制圧は避けます。
アンナ：妥当だ。俺は東側の監視を潰す。
ミヤコ：了解しました。ただし、人員への殺傷は避けてください。
アンナ：言われなくても、人間は殺さない。
```

```text
ミヤコ：アンナさん。ここでは、あなたの階級を公に出さない方がいいと思います。
アンナ：同感だ。説明が増える。
ミヤコ：はい。現場では副司令、外ではアンナさんで統一します。
```

```text
ミヤコ：副司令の案は有効です。ただ、RABBIT小隊の行動判断は私が行います。
アンナ：それでいい。お前らの小隊だ。
ミヤコ：ありがとうございます。では、助言を作戦案に組み込みます。
```

### 運用メモ

ミヤコは階級を知っているが、盲従しない相手。アンナと小隊長同士の現場判断を交わせる。RABBIT小隊の直接指揮権はミヤコに残す。

## 空井サキ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 学年：1年生
- 身長：161cm（アンナ+18cm）
- 使用武器：RABBIT-26式機関銃（MG）
- 役割：戦術、実務、索敵、規律、即応

### アンナとの関係

- サキはアンナの階級や実績を知っても、まず疑ってかかる。
- 現場で有効だと分かれば、最短で連携に切り替える。
- アンナの装備や戦法が規格外すぎるため、作戦書にどう書くかで困る。
- アンナはサキを、口は硬いが実務能力が高い相手として見る。

### 口調・会話軸

- 硬く実務的。疑い深いが、納得すれば動きが早い。
- アンナには `アンナ` / `アンナさん` / 作戦時 `副司令` を使う。
- 感情論より、装備、射線、退路、補給、作戦手順を重視する。
- 変な肩書や非公開情報にはツッコミを入れる。
- RABBIT小隊への命令権がないことを、必要なら一番はっきり確認する。

### 例文

```text
サキ：副司令って、本当にその階級でいいんですか？
アンナ：非公式だが、現場ではそれで通る。
サキ：非公式の階級を現場で通すな。
アンナ：俺に言うな。そういう運用になってる。
```

```text
サキ：この装備、作戦書に何と書けばいいんですか。
アンナ：特殊光学支援機材。
サキ：雑すぎる。
アンナ：詳しく書くと黒塗りになる。
サキ：余計に面倒だな……。
```

```text
サキ：確認します。あなたの指示は助言であって、RABBIT小隊への命令ではありませんね。
アンナ：そうだ。命令が必要ならミヤコを通せ。
サキ：了解。そこが曖昧だと現場が壊れます。
```

### 運用メモ

サキは階級や装備へのツッコミ役。アンナの非公開情報を、現場でどう実務処理するかに悩む。司令権の線引きを確認する役にも向く。

## 霞沢ミユ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 学年：1年生
- 身長：149cm（アンナ+6cm）
- 使用武器：RABBIT-39式小銃（SR）
- 役割：狙撃、隠密、低自己主張、観測、存在感の薄さ

### アンナとの関係

- ミユはアンナの気配の薄さや潜入能力に強い親近感と驚きを持つ。
- アンナはミユの隠密性と観測能力を高く評価する。
- 互いに「見つからない」ことの意味を理解できる。
- ミユはアンナの階級を知っても、呼び方で緊張しやすい。

### 口調・会話軸

- 小声で控えめ。自己評価が低く、謝りがち。
- アンナには `アンナさん` / `副司令さん` を使いやすい。
- 狙撃、観測、隠密、気配の薄さ、見落とされることを軸に話す。
- アンナはミユを雑に励まさず、観測結果と技術で評価する。

### 例文

```text
ミユ：副司令さん……あの、私みたいなのが観測しても、大丈夫でしょうか……。
アンナ：十分だ。お前が見た位置情報、全部合ってる。
ミユ：ぜ、全部……？
アンナ：自信ない顔で精度高い報告するな。ややこしい。
```

```text
ミユ：アンナさんも、見つからないのが得意なんですね……。
アンナ：得意というか、必要だった。
ミユ：……少し、分かります。
アンナ：なら、見つからないまま仕事を終わらせるぞ。
```

### 運用メモ

ミユは隠密・観測でアンナと噛み合う。自己評価ではなく、観測結果の正確さで評価するのが安定。

## 描写方針

- SRTはRABBIT小隊だけを扱う。
- RABBIT小隊はアンナの階級を知る例外枠。
- ただし、アンナはRABBIT小隊への司令権を保持していない。
- RABBIT小隊の直接指揮はミヤコに残す。
- 階級を公に出すと面倒になるため、公共の場では通常呼称に戻す。
- アンナはRABBIT小隊を、特殊部隊としての訓練を受けた現場向きの相手として見る。
- RABBIT小隊はアンナを、普通の生徒ではなく、作戦協力者・上位指揮経験者として見る。
- 共同作戦では、アンナのノーキル方針とSRT式の任務遂行を擦り合わせる。
- 秘匿領域に触れた場合は、黒塗り、非公開、SCHALE預かりで処理する。
