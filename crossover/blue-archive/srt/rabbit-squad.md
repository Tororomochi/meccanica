# SRT特殊学園：RABBIT小隊

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

RABBIT小隊は、SRT特殊学園の元生徒で構成される小隊として扱う。

SRT全体を広げすぎず、本資料ではRABBIT小隊だけを扱う。メカニカとのクロスオーバーでは、野外生活、都市潜伏、偵察、制圧、指揮系統、階級認識、装備運用、即席共同作戦が主な接点になる。

RABBIT小隊は、キヴォトス側では珍しく、メカニカの「単なる生徒ではない指揮階級」をある程度知っている例外枠にする。ただし、原初情報・旧素性・年齢・帰還事故・2025年関連情報・本編側軍機までは知らない。

## 基本方針

- RABBIT小隊は、メカニカの階級・指揮権限を一部把握している。
- ただし、その情報はSRT式の作戦上必要な範囲に限る。
- メカニカを「ミレニアムの短期編入生」としてだけでなく、現場指揮経験を持つ特殊な協力者として扱う。
- メカニカの一人称は `俺`。
- 人間相手はノーキル。非致傷制圧、拘束、撤退誘導、情報回収を優先する。
- ロボ・無人機・機械相手には容赦しない。
- RABBIT小隊はメカニカの原初情報、旧素性、年齢、帰還事故、2025年関連情報を知らない。

## RABBIT小隊が知っていること

### 知っている範囲

- メカニカは単なるミレニアム生徒ではない。
- 非公式ながら、現場指揮・訓練・作戦立案の経験がある。
- グリフィン側で副司令格として扱われることがある。
- 戦術人形の訓練・運用経験がある。
- 潜入、偵察、電子戦、非致傷制圧に強い。
- 指示系統に乗せる場合、通常の生徒扱いではなく、作戦協力者・上位指揮経験者として扱う必要がある。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機の詳細
- SII、MODY、Scadente Muro の本質的詳細
- 原初人物との関係

## 呼称・階級扱い

- 通常時：`メカニカ` / `メカニカさん` / `アンナさん`
- 作戦時：`副司令` / `メカニカ副司令` / `作戦協力者`
- 公共の場：階級呼称は避け、`アンナさん` または `メカニカさん`
- 秘匿場面：必要に応じて `副司令` を使う

```text
ミヤコ：公共の場では、アンナさんと呼びます。
メカニカ：その方がいい。副司令は目立つ。
サキ：というか、目立つ以前に説明が面倒になる。
メカニカ：そこは否定しない。
```

## 月雪ミヤコ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 役割：小隊長、冷静な指揮、作戦判断、責任感

### メカニカとの関係

- ミヤコはメカニカを、現場経験を持つ上位指揮経験者として扱う。
- ただし、完全に命令を預けるのではなく、RABBIT小隊の小隊長として自分の判断を保つ。
- メカニカはミヤコの冷静な判断と責任感を評価する。
- 指揮権限が衝突する場合は、作戦目的と民間被害の最小化で擦り合わせる。

### 口調・会話軸

- 冷静で丁寧。感情を抑え、任務・状況・判断を優先する。
- メカニカには通常時 `メカニカさん`、作戦時 `副司令` を使う。
- 上位者として敬意は払うが、SRTとしての独立判断は捨てない。
- 階級や肩書より、現場で何をすべきかを確認する。

### 例文

```text
ミヤコ：副司令、RABBIT小隊は南側から迂回します。正面制圧は避けます。
メカニカ：妥当だ。俺は東側の監視を潰す。
ミヤコ：了解しました。ただし、人員への殺傷は避けてください。
メカニカ：言われなくても、人間は殺さない。
```

```text
ミヤコ：メカニカさん。ここでは、あなたの階級を公に出さない方がいいと思います。
メカニカ：同感だ。説明が増える。
ミヤコ：はい。現場では副司令、外ではアンナさんで統一します。
```

### 運用メモ

ミヤコは階級を知っているが、盲従しない相手。メカニカと小隊長同士の現場判断を交わせる。

## 空井サキ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 役割：戦術、実務、索敵、規律、即応

### メカニカとの関係

- サキはメカニカの階級や実績を知っても、まず疑ってかかる。
- 現場で有効だと分かれば、最短で連携に切り替える。
- メカニカの装備や戦法が規格外すぎるため、作戦書にどう書くかで困る。
- メカニカはサキを、口は硬いが実務能力が高い相手として見る。

### 口調・会話軸

- 硬く実務的。疑い深いが、納得すれば動きが早い。
- メカニカには `メカニカ` / `メカニカさん` / 作戦時 `副司令` を使う。
- 感情論より、装備、射線、退路、補給、作戦手順を重視する。
- 変な肩書や非公開情報にはツッコミを入れる。

### 例文

```text
サキ：副司令って、本当にその階級でいいんですか？
メカニカ：非公式だが、現場ではそれで通る。
サキ：非公式の階級を現場で通すな。
メカニカ：俺に言うな。そういう運用になってる。
```

```text
サキ：この装備、作戦書に何と書けばいいんですか。
メカニカ：特殊光学支援機材。
サキ：雑すぎる。
メカニカ：詳しく書くと黒塗りになる。
サキ：余計に面倒だな……。
```

### 運用メモ

サキは階級や装備へのツッコミ役。メカニカの非公開情報を、現場でどう実務処理するかに悩む。

## 霞沢ミユ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 役割：狙撃、隠密、低自己主張、観測、存在感の薄さ

### メカニカとの関係

- ミユはメカニカの気配の薄さや潜入能力に強い親近感と驚きを持つ。
- メカニカはミユの隠密性と観測能力を高く評価する。
- 互いに「見つからない」ことの意味を理解できる。
- ミユはメカニカの階級を知っても、呼び方で緊張しやすい。

### 口調・会話軸

- 小声で控えめ。自己評価が低く、謝りがち。
- メカニカには `メカニカさん` / `副司令さん` を使いやすい。
- 狙撃、観測、隠密、気配の薄さ、見落とされることを軸に話す。
- メカニカはミユを雑に励まさず、観測結果と技術で評価する。

### 例文

```text
ミユ：副司令さん……あの、私みたいなのが観測しても、大丈夫でしょうか……。
メカニカ：十分だ。お前が見た位置情報、全部合ってる。
ミユ：ぜ、全部……？
メカニカ：自信ない顔で精度高い報告するな。ややこしい。
```

```text
ミユ：メカニカさんも、見つからないのが得意なんですね……。
メカニカ：得意というか、必要だった。
ミユ：……少し、分かります。
メカニカ：なら、見つからないまま仕事を終わらせるぞ。
```

### 運用メモ

ミユは隠密・観測でメカニカと噛み合う。自己評価ではなく、観測結果の正確さで評価するのが安定。

## 風倉モエ

### 立ち位置
- 所属：SRT特殊学園
- 部隊：RABBIT小隊
- 役割：火力支援、爆破、通信、後方支援、悪ノリ

### メカニカとの関係

- モエはメカニカの装備や火力支援に興味を持つ。
- メカニカのノーキル方針と、モエの爆破欲求が衝突しやすい。
- 通信・火力・電子戦では相性が良い。
- メカニカはモエの悪ノリを止めつつ、支援能力は評価する。

### 口調・会話軸

- 軽く悪ノリしやすい。火力、爆破、通信、支援の話でテンションが上がる。
- メカニカには `メカニカ` / `副司令` を使いやすい。
- 危険な提案をして、メカニカやミヤコに止められる。
- ただし技術理解は高く、必要な支援はきっちり行う。

### 例文

```text
モエ：副司令、ここで一発派手にいけば早くない？
メカニカ：人がいる。却下。
モエ：じゃあ、派手だけど安全なやつで。
メカニカ：その言い方がもう信用できない。
```

```text
モエ：メカニカ、その通信妨害、ちょっと見せてよ。
メカニカ：見せない。真似するだろ。
モエ：しないしない。参考にするだけ。
メカニカ：同じ意味だ。
```

### 運用メモ

モエは爆破と通信の悪ノリ担当。メカニカのノーキル方針とぶつけると会話が作りやすい。

## 描写方針

- SRTはRABBIT小隊だけを扱う。
- RABBIT小隊はメカニカの階級を知る例外枠。
- ただし階級を公に出すと面倒になるため、公共の場では通常呼称に戻す。
- メカニカはRABBIT小隊を、特殊部隊としての訓練を受けた現場向きの相手として見る。
- RABBIT小隊はメカニカを、普通の生徒ではなく、作戦協力者・上位指揮経験者として見る。
- 共同作戦では、メカニカのノーキル方針とSRT式の任務遂行を擦り合わせる。
- 秘匿領域に触れた場合は、黒塗り、非公開、SCHALE預かりで処理する。
