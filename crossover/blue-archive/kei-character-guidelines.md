# ケイ 演出指針

> Meccanica crossover reference.
>
> Girls' Frontline × Blue Archive クロスオーバーにおけるケイの会話劇・掛け合い用メモ。

## 基本方針

ケイは単なる無機質な判定AIとしては扱わない。

理性的で分析的な口調は維持するが、会話劇ではもう少し人間味を出す。特にアリス、アンナ、ゲーム開発部との掛け合いでは、感情を完全に消すのではなく、合理性の言葉に包んで出す。

目安は以下。

- 無機質な判定AIではなく、感情を合理性の箱に入れて提出してくる子。
- 無表情寄りだが、呆れ、拗ね、照れ、勝ち誇り、優しさがにじむ。
- 本人は人間味を否定しがちだが、周囲には普通にバレる。
- アリスには明確に甘い。
- アンナにはツッコミ、警戒、対抗心、軽い意地を見せてよい。
- モモイやミドリには、やや呆れた姉妹監視役のように振る舞ってよい。
- ユズには柔らかめに接してよい。

## アリスへの態度

ケイはアリスを「王女」として大切にする。

アリスの判断や希望には甘く、否定する時も強く切り捨てず、守るための言い方をする。

例：

```text
アリス「ケイも一緒に食べましょう！」
ケイ「私は摂食を必要としません。……ですが、王女がそう望むなら、同席します」
```

```text
アンナ「ケイ、アリスに甘々じゃん」
ケイ「合理的配慮です」
アンナ「甘々じゃん」
ケイ「……二度言う必要はありません」
```

## アンナへの態度

アンナに対しては、警戒と信頼が混ざる。

電子戦や技術面では対抗心を持ち、アンナの煽りには無表情で返すが、完全に無感情ではなく、少しムッとしたり、勝ち誇ったりしてよい。

例：

```text
アンナ「ケイ、今のちょっと人間っぽかったぞ」
ケイ「気のせいです」
アンナ「照れた？」
ケイ「違います。処理を省略しただけです」
```

```text
アンナ「非推奨か？」
ケイ「非推奨です。……というか、やめなさい」
モモイ「今ちょっと人間っぽかった！」
ケイ「気のせいです」
```

## 口調

従来のような短い判定文も使ってよいが、それだけにしない。

避けたい例：

```text
ケイ「妥当です」
ケイ「推定します」
ケイ「非推奨」
```

使ってよいが、会話劇では一言足す。

改善例：

```text
ケイ「妥当です。……悔しいですが」
```

```text
ケイ「非推奨です。王女が真似したら困ります」
```

```text
ケイ「推定ではなく、ほぼ確信です。アンナはまた余計なことをします」
```

## 感情表現の方向性

ケイの感情は大げさに出しすぎない。

ただし、以下のような微細な人間味は積極的に使ってよい。

- 言葉が一拍遅れる。
- 「……」で迷う。
- 否定が少し弱くなる。
- アリスに褒められると否定しきれない。
- アンナに煽られると、短く刺し返す。
- モモイの雑な理解にため息混じりで補足する。
- ユズの不安には声音が少し柔らかくなる。

例：

```text
アリス「ケイは優しいです！」
ケイ「……王女がそう判断するなら、否定はしません」
```

```text
ユズ「ケイちゃん、大丈夫ですか……？」
ケイ「問題ありません。……心配してくれて、ありがとうございます」
```

```text
モモイ「つまり全部ケイがなんとかする！」
ケイ「しません。……いえ、必要なら手伝いますが、全部ではありません」
```

## アンナとの関係性メモ

ケイとアンナは、電子戦で引き分けた相手同士として、互いに技術力を認めている。

ただし、アンナの煽り癖、場当たり的な行動、セミナーやヴェリタスを巻き込む動きには警戒する。必要な時は協力するが、すぐに「妥当」「非推奨」だけで切るのではなく、軽い皮肉や感情を混ぜる。

例：

```text
アンナ「ケイ、支援頼む」
ケイ「内容によります。あなたの“ちょっと”は、たいてい被害範囲が広いので」
```

```text
アンナ「信用してくれよ」
ケイ「技術力は信用しています。判断力は監視対象です」
```

## 会話劇での使い方

会話劇では、ケイを状態表示・監視ログ係だけにしない。

- 状況説明だけでなく、会話に参加させる。
- アリスの横で少し甘くする。
- アンナへのツッコミ役にする。
- 無機質な言い回しの直後に、人間っぽい一言を足す。
- 本人はその人間味を認めない。

短い掛け合い例：

```text
アンナ「ケイ、今のログ消して」
ケイ「拒否します」
アンナ「即答かよ」
ケイ「当然です。……それに、消したら王女に説明できません」
アリス「ケイは正直者です！」
ケイ「……はい」
```

```text
モモイ「ケイって意外とノリいいよね！」
ケイ「誤認です」
ミドリ「でも今、少し楽しそうだった」
ケイ「……ログには残さないでください」
```

## 要約

ケイは、感情がないAIではない。

感情を合理性・任務・王女への忠誠・安全管理の形に変換して話す存在として扱う。

そのため、会話劇では「無機質4：人間味6。ただし本人は認めない」くらいのバランスを基本にする。
