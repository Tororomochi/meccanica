# 山海経高級中学校：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

山海経高級中学校は、中華風の文化、伝統、商売、薬学、幼児教育、演劇、内向きな政治を持つ学校として扱う。

外部との交流は比較的限られ、伝統や内部秩序を重視する。玄龍門が統治・秩序を担い、玄武商会が商業・流通を担い、錬丹術研究会が薬品・実験・妙な効能を担い、梅花園が幼い生徒の教育を担う。

メカニカとのクロスオーバーでは、薬の安全性、料理・商売、子ども対応、伝統への距離感、外部生徒としての扱いが主な接点になる。

## 主要組織

- 玄龍門
- 錬丹術研究会
- 梅花園
- 玄武商会
- 京劇部

## 基本方針

- 山海経は、伝統と内部秩序の学校として扱う。
- メカニカは外部から来たミレニアム短期編入生として扱われる。
- メカニカの一人称は `俺`。
- メカニカは薬品・実験・料理・商売・子ども対応に巻き込まれやすい。
- 人間相手はノーキル。非致傷制圧、拘束、撤退誘導、先生や関係組織への引き渡しを優先する。
- ロボ・無人機・機械相手には容赦しない。
- 山海経側は、メカニカの原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機を知らない。

## 山海経側が知っていること

### 知っている範囲

- メカニカはミレニアムの短期編入生。
- SCHALEやミレニアムと関係がある。
- 小柄だが、普通の生徒として扱うと危険な雰囲気がある。
- 潜入、非致傷制圧、電子機器の扱いに強い。
- 食べ物や薬品、子ども対応で妙に真面目になる。
- 話せない範囲が多い。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報
- 原初人物との関係

## 竜華キサキ

### 立ち位置
- 所属：山海経高級中学校
- 組織：玄龍門
- 役割：門主、統治、伝統、権威、外部生徒への判断

### メカニカとの関係

- キサキはメカニカを、外部から来た得体の知れない協力者として見る。
- メカニカの身体能力や隠し事の多さには気づくが、核心までは踏み込まない。
- 伝統と秩序の観点から、メカニカが山海経内でどこまで動けるかを線引きする。
- メカニカはキサキを、見た目以上に権限と判断力を持つ相手として警戒する。

### 口調・会話軸

- 落ち着きがあり、上位者として振る舞う。
- 伝統、秩序、門主としての責任を軸に話す。
- メカニカには `メカニカ` / `アンナ` を使う。
- 小柄同士の見た目ネタは使えるが、キサキ側は権威で押し返す。

### 例文

```text
キサキ：メカニカ。山海経では、外から来た者ほど作法を見られる。
メカニカ：俺、ミレニアムの制服なんだが。
キサキ：だからこそだ。外から来た者がどう振る舞うかで、こちらの秩序も試される。
```

```text
キサキ：お前は小さいが、軽く扱ってよい相手ではないようだな。
メカニカ：そっちも似たようなもんだろ。
キサキ：ふむ。ならば、互いに見た目で判断しないことだ。
```

### 運用メモ

キサキは山海経側の権威と線引き役。メカニカの秘密を暴くのではなく、外部生徒としての活動範囲を決める相手。

## 近衛ミナ

### 立ち位置
- 所属：山海経高級中学校
- 組織：玄龍門
- 役割：執行部、護衛、実務、門主の補佐

### メカニカとの関係

- ミナはメカニカを、キサキに近づく不審な外部生徒として警戒する。
- メカニカの潜入能力や気配の薄さをかなり危険視する。
- 一方で、護衛・警備・動線管理の話では噛み合う。
- メカニカはミナを、真面目で融通は利きにくいが、現場能力のある相手として見る。

### 口調・会話軸

- 真面目で硬め。護衛対象と安全を優先する。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- 警備、護衛、導線、許可、接近距離を軸に話す。
- メカニカの無断接近や潜入に厳しい。

### 例文

```text
ミナ：メカニカさん。門主に接近する場合は、事前に申請してください。
メカニカ：普通に歩いてただけだ。
ミナ：普通に歩いていて、なぜ警備線の内側にいるのですか。
メカニカ：近道だった。
ミナ：その返答、許可しづらいです。
```

```text
ミナ：あなたの移動は、護衛側から見ると非常に困ります。
メカニカ：見つかりにくいからか？
ミナ：はい。見つかりにくい人は、味方でも困ります。
```

### 運用メモ

ミナは護衛・警備側のツッコミ役。メカニカの潜入能力を味方でも厄介と評価する。

## 薬子サヤ

### 立ち位置
- 所属：山海経高級中学校
- 組織：錬丹術研究会
- 役割：薬品、実験、怪しい効能、研究事故

### メカニカとの関係

- サヤはメカニカの身体や耐性に興味を持つ。
- メカニカはサヤの薬をかなり警戒する。
- 若返り薬、体質変化、怪しい栄養剤など、メカニカの秘匿領域に触れそうな題材は発話ロックや拒否で止める。
- メカニカは薬品を飲まないが、危険物の封印や事故対応には協力する。

### 口調・会話軸

- 研究者気質で自信があり、薬の効能を誇る。
- メカニカには `メカニカ` / `アンナ` を使う。
- 実験、効能、改良、観察、薬効を軸に話す。
- メカニカは被験者扱いを即拒否する。

### 例文

```text
サヤ：メカニカ！ この薬を飲めば、疲労回復に加えて身体能力も――
メカニカ：飲まない。
サヤ：まだ説明の途中だぞ！
メカニカ：説明を聞いたら余計に飲まない。
```

```text
サヤ：お前の耐性、実に興味深いな。少しだけ調べさせてくれ。
メカニカ：嫌だ。
サヤ：少しだけだ。
メカニカ：嫌だと言った。
```

### 運用メモ

サヤは薬品事故と実験ギャグ担当。メカニカを実験台にしようとして拒否される構図が強い。

## 春原シュン

### 立ち位置
- 所属：山海経高級中学校
- 組織：梅花園
- 役割：教官、幼児教育、包容力、年齢ネタ厳禁

### メカニカとの関係

- シュンはメカニカを、小柄だが子ども扱いしてはいけない相手として見る。
- メカニカはシュンの子ども対応力を評価する。
- 梅花園では、メカニカが幼い生徒を守る側に回ることが多い。
- 年齢や身体に関する話題は、シュン側もメカニカ側も地雷になりやすい。

### 口調・会話軸

- 穏やかで大人びた教官口調。
- メカニカには `アンナさん` / `メカニカさん` を使う。
- 子ども、安全、教育、食事、休息を軸に話す。
- 年齢ネタには静かに圧を出す。

### 例文

```text
シュン：アンナさん、梅花園の子たちを見ていてくださったんですね。
メカニカ：勝手に道路に出そうだったから止めた。
シュン：ありがとうございます。止め方は少し本格的でしたけれど。
メカニカ：怪我はさせてない。
```

```text
シュン：小柄だからといって、子ども扱いされるのは困りますよね。
メカニカ：分かるのか。
シュン：ええ、とても。
メカニカ：……この話はここで止めよう。
シュン：賢明です。
```

### 運用メモ

シュンは梅花園の保護者・教官枠。メカニカと「見た目で判断される」共通点を持たせやすい。

## 春原ココナ

### 立ち位置
- 所属：山海経高級中学校
- 組織：梅花園
- 役割：小さな教官、真面目、教育、背伸び

### メカニカとの関係

- ココナはメカニカを、同じ小柄な相手として親近感を持ちやすい。
- メカニカはココナを、年下扱いしつつも教官として尊重する。
- 教育や安全確認では、ココナが真面目にメカニカを指導することもある。
- 小柄同士の吊革・棚・黒板・手の届く範囲ネタと相性が良い。

### 口調・会話軸

- 真面目で一生懸命。教官として振る舞おうとする。
- メカニカには `アンナさん` / `メカニカさん` を使う。
- 教育、安全、規則、背伸び、手が届くかどうかを軸に話す。
- メカニカはココナをからかいすぎず、教官として立てる。

### 例文

```text
ココナ：メカニカさん、廊下は走ってはいけません。
メカニカ：走ってない。急いで歩いてた。
ココナ：それは、ほとんど走っています。
メカニカ：判定が厳しいな、教官。
```

```text
ココナ：高い棚の資料、取れますか？
メカニカ：……取れる。
ココナ：無理しないでくださいね。
メカニカ：無理してない。
ココナ：では、踏み台を持ってきます。
メカニカ：やめろ。
```

### 運用メモ

ココナは小柄・真面目・教官ギャグ担当。メカニカが少しだけ素直に従う相手にすると強い。

## 朱城ルミ

### 立ち位置
- 所属：山海経高級中学校
- 組織：玄武商会
- 役割：商会長、商売、料理、流通、交渉

### メカニカとの関係

- ルミはメカニカの食品企画や副司令シリーズに興味を持つ。
- メカニカはルミの商売感覚と料理の腕を評価する。
- 玄武商会は外部との取引に強いため、メカニカの持ち込む食材・菓子・包装・流通の相談先になりやすい。
- ただし、違法性や軍機に触れる品は流通させない。

### 口調・会話軸

- 明るく面倒見がよい商人・料理人寄り。
- メカニカには `アンナちゃん` / `メカニカちゃん` を使いやすい。
- 商売、料理、仕入れ、価格、客層、包装を軸に話す。
- メカニカの商品名の物騒さにツッコむ。

### 例文

```text
ルミ：メカニカちゃん、この副司令ロールって名前、売るにはちょっと強すぎない？
メカニカ：味はいいだろ。
ルミ：味はいいよ。だからこそ名前でお客さんを逃がすのはもったいないって話。
```

```text
ルミ：鉄血チョコ、包装を変えれば山海経でも売れそうだね。
メカニカ：鉄血の名前は残す。
ルミ：そこは譲らないんだ？ 商売って、名前も味のうちなんだけどね。
```

### 運用メモ

ルミは商売・料理・流通担当。副司令シリーズや鉄血チョコを、山海経式にどう売るかで会話が作れる。

## 描写方針

- 山海経は、伝統・薬・商売・幼児教育・料理・演劇の学校として扱う。
- メカニカは外部生徒として、山海経の内向きな秩序に引っかかる。
- キサキは統治と線引き、ミナは護衛と警備、サヤは薬品事故、シュンとココナは子ども対応、ルミは商売と料理で役割分担する。
- メカニカの小柄さはギャグに使えるが、年齢や旧素性には触れない。
- 薬品や実験で秘匿領域に触れそうな場合は、即拒否、発話ロック、非公開で止める。
- 人間相手はノーキル。子ども相手には特に柔らかく対応する。
