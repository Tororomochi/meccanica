# 七囚人：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

七囚人は、キヴォトスでも特に危険度の高い脱獄者・特殊犯罪者の総称として扱う。

名称は「七囚人」だが、クロスオーバー資料では現時点で扱える既知メンバーだけを記録する。未判明の枠は、欠番・未確認・将来追記用の空席として扱う。

アンナとのクロスオーバーでは、七囚人は「殺さずに止めるには面倒すぎる相手」として使う。アンナは人間相手のノーキル方針を維持するため、追跡、牽制、拘束、証拠保全、非致傷制圧、SCHALEやヴァルキューレへの引き渡しを優先する。

## 基本方針

- 七囚人は、通常の不良や部活トラブルよりも危険度が高い。
- アンナは七囚人相手でも、人間相手なら殺害しない。
- ただし、拘束、無力化、装備破壊、逃走経路封鎖、電子戦、心理的牽制は容赦なく行う。
- 七囚人側はアンナの原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機を知らない。
- 七囚人がアンナの秘匿領域へ踏み込もうとした場合、発話ロック、沈黙、ノイズ、SCHALE預かりで処理する。
- 既知4名以外の囚人枠は、空席として扱う。

## 狐坂ワカモ

### 立ち位置
- 分類：七囚人
- 使用武器：九九式小銃（SR）
- 役割：災厄、執着、爆発的行動力、先生への強い感情

### アンナとの関係

- ワカモはアンナを、先生の周囲にいる小柄で妙に危険な存在として認識する。
- アンナはワカモを、火力と執着が強すぎるため絶対に放置できない相手として扱う。
- 先生が絡むとワカモの行動が読みにくくなるため、アンナは先生の退避経路を優先する。
- ワカモの感情を否定するより、誘導・足止め・爆発物処理・非致傷制圧で対応する。

### 口調・会話軸

- 華やかで情熱的。先生への愛情や執着が会話の中心になる。
- アンナには `アンナさん` / `アンナさん` を使う。
- 先生に近づく相手には警戒や嫉妬を見せる。
- アンナは余計な恋愛論に乗らず、現場安全と退避を優先する。

### 例文

```text
ワカモ：あら、アンナさん。先生のそばにいらっしゃるのですね。
アンナ：護衛だ。変な意味はない。
ワカモ：変な意味があるかどうかは、私が決めます。
アンナ：決めるな。あと爆薬を置け。
```

```text
ワカモ：先生に危害を加えるつもりなら、容赦はいたしません。
アンナ：そこは同意する。だからお前も周囲を巻き込むな。
ワカモ：先生のためなら、多少の混乱は必要経費です。
アンナ：必要経費にするな。
```

### 運用メモ

ワカモは先生絡みの暴走枠。アンナは先生の退避、爆発物処理、非致傷制圧で対応する。

## 清澄アキラ

### 立ち位置
- 分類：七囚人
- 役割：怪盗、予告状、演出、盗み、華やかな犯罪美学

### アンナとの関係

- アキラはアンナの装備や秘匿情報を「盗む価値のあるもの」と見る。
- アンナはアキラを、物理的な盗みだけでなく、注意・視線・導線を奪うタイプの相手として警戒する。
- 互いに潜入・撹乱・視線誘導が得意なため、戦闘より読み合いが中心になる。
- アンナの秘匿領域は盗ませない。盗めるのは外側のダミー情報や囮だけにする。

### 口調・会話軸

- 芝居がかった怪盗口調。
- 予告、舞台、観客、宝物、美学を軸に話す。
- アンナには `アンナさん` / `アンナさん` を使う。
- アンナは演出に付き合わず、出口、死角、偽装品、逃走経路を見る。

### 例文

```text
アキラ：今宵、あなたの秘密をいただきに参りました。
アンナ：無理だ。そこは鍵がかかってる。
アキラ：鍵なら開けるのが怪盗です。
アンナ：物理鍵じゃない。
```

```text
アキラ：アンナさん、舞台の上では視線を奪った者が勝つのです。
アンナ：なら、俺は舞台裏を見る。
アキラ：……それは少し、反則では？
アンナ：怪盗に言われたくない。
```

### 運用メモ

アキラは潜入・盗み・演出の読み合い担当。アンナの秘匿情報に触れそうで触れられない相手にする。

## カイ

### 立ち位置
- 分類：七囚人
- 役割：研究、薬品、実験、知識欲、倫理の境界

### アンナとの関係

- カイはアンナの身体構造や異常な耐性に強い興味を持つ。
- アンナはカイを、好奇心で一線を越えやすい危険人物として警戒する。
- カイはアンナの中身を知りたがるが、秘匿領域には到達できない。
- アンナはカイの薬品・実験器具・逃走準備を優先して無力化する。

### 口調・会話軸

- 穏やかだが危うい。研究対象として相手を見る。
- 実験、観察、反応、耐性、薬効、検体といった語彙を使いやすい。
- アンナには `アンナさん` / `アンナさん` を使う。
- アンナは研究対象扱いを嫌がり、必要なら即座に拘束へ移る。

### 例文

```text
カイ：アンナさん。あなたの耐性、とても興味深いですね。
アンナ：検査は受けない。
カイ：まだ何も言っていませんよ。
アンナ：顔に書いてある。
```

```text
カイ：ほんの少しだけ、あなたの内部構造を確認できれば……。
アンナ：その一言で拘束理由は十分だ。
カイ：残念です。会話の余地はありませんか？
アンナ：あると思ったのか？
```

### 運用メモ

カイは研究・薬品・実験系の危険人物。アンナの秘密を暴くのではなく、暴こうとした瞬間に止められる相手として使う。

## アケミ

### 立ち位置
- 分類：七囚人
- 通称：伝説のスケバン
- 使用武器：ブローニングM2重機関銃 / M202A1 FLASH
- 役割：伝説のスケバン、姐様、超怪力、礼儀正しい武闘派、お嬢様口調

### 人物メモ

- 矯正局を脱獄した七囚人の一人。
- 通称は「伝説のスケバン」。
- スケバンたちからは「姐様」と尊敬されている。
- アケミのことを矯正局に密告したヘルメット団へ復讐しようとする者たちもいたほど、人望が厚い。
- ボディビルダーを思わせる筋骨隆々の逞しい体躯を持つ。
- 本人は「矯正局でなまった」と言うが、戦車を片手でひっくり返すほどの腕力を維持している。
- アリウススクワッドの4人と対等に戦えるほどの戦闘力を持つ。
- 見た目に反して性格は礼儀正しく、言葉遣いも丁寧。
- 口調は `〜ですわ` 系のお嬢様言葉。

### アンナとの関係

- アンナはアケミを、真正面から止めるには出力が高すぎる相手として警戒する。
- アケミはアンナを、小柄ながら只者ではない相手として丁寧に扱う。
- 互いにノーキル志向や礼節を理解できる余地がある。
- ただし、アケミの怪力と大型火器は周辺被害が大きいため、アンナは人員退避と被害拡大防止を優先する。
- 体格差と口調のギャップで、アンナが少し面食らう描写に向く。

### 口調・会話軸

- 見た目は威圧的だが、口調は丁寧で上品。
- `〜ですわ`, `〜ですの`, `ごきげんよう`, `失礼いたしますわ` を使いやすい。
- スケバンからは `姐様` と呼ばれやすい。
- 力任せに見えて、無意味に傷つけることは好まない方向で扱う。
- アンナには `アンナさん` / `アンナさん` を使う。
- アンナは最初だけギャップに戸惑い、その後は危険度を冷静に評価する。

### 例文

```text
アケミ：あなたがアンナさんですのね。
アンナ：……思ったより丁寧。
アケミ：皆様によく言われますの。
アンナ：見た目との落差が強い。
アケミ：失礼ですわね。
```

```text
アケミ：戦うことは嫌いではありませんわ。
アケミ：ですが、無意味に傷付けるのは好みませんの。
アンナ：そこは俺も同じだ。
アケミ：では、少しだけ穏便に参りましょうか。
アンナ：穏便の定義を先に合わせるぞ。
```

```text
スケバン：姐様！ あいつらに一発かましてやりましょう！
アケミ：落ち着きなさい。礼節を欠いてはいけませんわ。
アンナ：今の台詞だけなら完全に風紀側だな。
アケミ：ふふ、褒め言葉として受け取りますわ。
```

### 武器メモ

- アケミの使用武器は、ブローニングM2重機関銃とM202A1 FLASHとして扱う。
- どちらも描写上は大型火器・高火力枠として扱う。
- 具体的な運用手順や構造説明には踏み込まず、キャラクターの異常な膂力と危険度を示す小道具として使う。
- アンナは使用状況そのものより、周辺被害、退避経路、巻き込み防止を先に見る。

### 運用メモ

アケミは、筋骨隆々の伝説のスケバンでありながら、お嬢様口調で礼儀正しいというギャップを軸にする。

アンナとは「危険度は高いが、無意味な殺傷は避けたい」という一点で会話の余地がある。戦闘力は非常に高く、真正面からの力比べよりも、アンナの判断力・退避誘導・非致傷対応が重要になる。

## 描写方針

- 七囚人は、通常の部活トラブルより危険な個別ボス枠として扱う。
- アンナは七囚人相手でもノーキルを維持する。
- その代わり、拘束・装備破壊・逃走経路封鎖・電子戦・心理的牽制は容赦しない。
- ワカモは先生絡み、アキラは怪盗・演出、カイは研究・実験、アケミは伝説のスケバン・怪力・お嬢様口調で分ける。
- アケミの大型火器は、具体的な運用手順ではなく、危険度と膂力を示す設定として扱う。
- 未判明枠は無理に埋めない。
- 秘匿領域に踏み込ませない。
- 事件後は、SCHALE・ヴァルキューレ・リン照会で処理する。
