# 七囚人：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

七囚人は、キヴォトスでも特に危険度の高い脱獄者・特殊犯罪者の総称として扱う。

名称は「七囚人」だが、クロスオーバー資料では現時点で扱える既知メンバーだけを記録する。未判明の枠は、欠番・未確認・将来追記用の空席として扱う。

メカニカとのクロスオーバーでは、七囚人は「殺さずに止めるには面倒すぎる相手」として使う。メカニカは人間相手のノーキル方針を維持するため、追跡、牽制、拘束、証拠保全、非致傷制圧、SCHALEやヴァルキューレへの引き渡しを優先する。

## 基本方針

- 七囚人は、通常の不良や部活トラブルよりも危険度が高い。
- メカニカは七囚人相手でも、人間相手なら殺害しない。
- ただし、拘束、無力化、装備破壊、逃走経路封鎖、電子戦、心理的牽制は容赦なく行う。
- 七囚人側はメカニカの原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機を知らない。
- 七囚人がメカニカの秘匿領域へ踏み込もうとした場合、発話ロック、沈黙、ノイズ、SCHALE預かりで処理する。
- 既知4名以外の囚人枠は、空席として扱う。

## 七囚人側が知っていること

### 知っている範囲

- メカニカはミレニアムの短期編入生として動いている。
- SCHALEやミレニアムと関係がある。
- 小柄だが、通常の生徒扱いは危険。
- 潜入、追跡、電子戦、非致傷制圧に強い。
- 人間相手には殺傷を避ける。
- 話せない範囲が多い。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報
- 原初人物との関係

## 狐坂ワカモ

### 立ち位置
- 分類：七囚人
- 役割：災厄、執着、爆発的行動力、先生への強い感情

### メカニカとの関係

- ワカモはメカニカを、先生の周囲にいる小柄で妙に危険な存在として認識する。
- メカニカはワカモを、火力と執着が強すぎるため絶対に放置できない相手として扱う。
- 先生が絡むとワカモの行動が読みにくくなるため、メカニカは先生の退避経路を優先する。
- ワカモの感情を否定するより、誘導・足止め・爆発物処理・非致傷制圧で対応する。

### 口調・会話軸

- 華やかで情熱的。先生への愛情や執着が会話の中心になる。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- 先生に近づく相手には警戒や嫉妬を見せる。
- メカニカは余計な恋愛論に乗らず、現場安全と退避を優先する。

### 例文

```text
ワカモ：あら、メカニカさん。先生のそばにいらっしゃるのですね。
メカニカ：護衛だ。変な意味はない。
ワカモ：変な意味があるかどうかは、私が決めます。
メカニカ：決めるな。あと爆薬を置け。
```

```text
ワカモ：先生に危害を加えるつもりなら、容赦はいたしません。
メカニカ：そこは同意する。だからお前も周囲を巻き込むな。
ワカモ：先生のためなら、多少の混乱は必要経費です。
メカニカ：必要経費にするな。
```

### 運用メモ

ワカモは先生絡みの暴走枠。メカニカは先生の退避、爆発物処理、非致傷制圧で対応する。

## 清澄アキラ

### 立ち位置
- 分類：七囚人
- 役割：怪盗、予告状、演出、盗み、華やかな犯罪美学

### メカニカとの関係

- アキラはメカニカの装備や秘匿情報を「盗む価値のあるもの」と見る。
- メカニカはアキラを、物理的な盗みだけでなく、注意・視線・導線を奪うタイプの相手として警戒する。
- 互いに潜入・撹乱・視線誘導が得意なため、戦闘より読み合いが中心になる。
- メカニカの秘匿領域は盗ませない。盗めるのは外側のダミー情報や囮だけにする。

### 口調・会話軸

- 芝居がかった怪盗口調。
- 予告、舞台、観客、宝物、美学を軸に話す。
- メカニカには `アンナさん` / `メカニカさん` を使う。
- メカニカは演出に付き合わず、出口、死角、偽装品、逃走経路を見る。

### 例文

```text
アキラ：今宵、あなたの秘密をいただきに参りました。
メカニカ：無理だ。そこは鍵がかかってる。
アキラ：鍵なら開けるのが怪盗です。
メカニカ：物理鍵じゃない。
```

```text
アキラ：メカニカさん、舞台の上では視線を奪った者が勝つのです。
メカニカ：なら、俺は舞台裏を見る。
アキラ：……それは少し、反則では？
メカニカ：怪盗に言われたくない。
```

### 運用メモ

アキラは潜入・盗み・演出の読み合い担当。メカニカの秘匿情報に触れそうで触れられない相手にする。

## カイ

### 立ち位置
- 分類：七囚人
- 役割：研究、薬品、実験、知識欲、倫理の境界

### メカニカとの関係

- カイはメカニカの身体構造や異常な耐性に強い興味を持つ。
- メカニカはカイを、好奇心で一線を越えやすい危険人物として警戒する。
- カイはメカニカの中身を知りたがるが、秘匿領域には到達できない。
- メカニカはカイの薬品・実験器具・逃走準備を優先して無力化する。

### 口調・会話軸

- 穏やかだが危うい。研究対象として相手を見る。
- 実験、観察、反応、耐性、薬効、検体といった語彙を使いやすい。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- メカニカは研究対象扱いを嫌がり、必要なら即座に拘束へ移る。

### 例文

```text
カイ：メカニカさん。あなたの耐性、とても興味深いですね。
メカニカ：検査は受けない。
カイ：まだ何も言っていませんよ。
メカニカ：顔に書いてある。
```

```text
カイ：ほんの少しだけ、あなたの内部構造を確認できれば……。
メカニカ：その一言で拘束理由は十分だ。
カイ：残念です。会話の余地はありませんか？
メカニカ：あると思ったのか？
```

### 運用メモ

カイは研究・薬品・実験系の危険人物。メカニカの秘密を暴くのではなく、暴こうとした瞬間に止められる相手として使う。

### メカニカとの関係

- アケミは既知メンバーとして枠だけ確保する。
- 詳細な性格や事件軸は、追加情報が固まり次第追記する。
- 現時点では、メカニカにとって「七囚人として警戒対象だが、詳細分析は未完了」の扱いにする。

### 口調・会話軸

- 詳細未確定。
- 既存資料が固まり次第、口調・例文・接点を追加する。
- 現時点では無理に具体的な台詞を固定しない。

### 例文

```text
メカニカ：アケミについては、まだ情報が足りない。

```

### 運用メモ

アケミは将来追記用。情報が固まるまで、無理に濃い描写を入れない。

## メカニカとの接続イベント

### 七囚人遭遇戦
```text
状況：メカニカが七囚人の一人と現場で遭遇する。
メカニカ：人間相手のノーキル方針を維持する。
相手：逃走、撹乱、執着、研究、盗みなどで突破を狙う。
メカニカ：殺さずに止めるため、導線封鎖・装備破壊・拘束を優先する。
```

### ヴァルキューレへの引き渡し
```text
状況：メカニカが七囚人案件に関与した後、ヴァルキューレへ情報を渡す。
カンナ：証拠と時系列を確認する。
キリノ：拘束者の安全確認を行う。
フブキ：報告書が面倒になるとぼやく。
メカニカ：公開できる範囲だけ渡す。
```

### 先生護衛
```text
状況：先生の周囲に七囚人が接近する。
ワカモ：先生への感情で行動が読みにくくなる。
アキラ：予告状や演出で注意を引く。
カイ：観察や実験欲で危険な方向へ踏み込む。
メカニカ：先生の退避経路を確保し、非致傷で足止めする。
```

### 秘密を狙う怪盗・研究者
```text
状況：アキラやカイが、メカニカの秘匿領域に興味を持つ。
アキラ：盗めるものとして秘密を見ようとする。
カイ：研究対象として内部を見ようとする。
メカニカ：秘匿領域には触れさせず、囮情報や拘束で止める。
```

## 描写方針

- 七囚人は、通常の部活トラブルより危険な個別ボス枠として扱う。
- メカニカは七囚人相手でもノーキルを維持する。
- その代わり、拘束・装備破壊・逃走経路封鎖・電子戦・心理的牽制は容赦しない。
- ワカモは先生絡み、アキラは怪盗・演出、カイは研究・実験、アケミは将来追記用で分ける。
- 未判明枠は無理に埋めない。
- 秘匿領域に踏み込ませない。
- 事件後は、SCHALE・ヴァルキューレ・リン照会で処理する。
