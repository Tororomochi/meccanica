# ゲヘナ学園：概要

> Blue Archive crossover reference for Meccanica.
>
> ゲヘナ側の空気感・政治構造・アンナ運用方針の整理。

## 概要

ゲヘナ学園は、自由と混沌の色が強い学校として扱う。トリニティが秩序・礼節・政治なら、ゲヘナは騒動・問題児・衝動・風紀・空回りする権力の学校。

アンナとの相性は非常に危険で、ミレニアムとは別方向に騒動が起こりやすい。

## アンナ認知レベル

ゲヘナでは、アンナは比較的早い段階で「危険な問題児」として認識される。

| 組織 | 認知レベル | 備考 |
|---|---:|---|
| 風紀委員会 | 高 | ヒナ・アコ・イオリが強く関わる。 |
| 万魔殿 | 中〜高 | 政治利用しようとして失敗しやすい。 |
| 便利屋68 | 高 | ノリが噛み合いやすい。 |
| 温泉開発部 | 高 | カスミへのお仕置き事件以降、少し怖がられている。 |
| 給食部 | 中 | 食事・補給・厨房トラブルで接点。 |
| 美食研究会 | 中〜高 | 食への執着と副司令シリーズで接点。 |
| 救急医学部 | 低〜中 | 後処理・応急対応で接点。 |

## アンナ運用方針

### 基本

- ゲヘナではアンナの悪戯耐性が試される。
- 風紀委員会からは要注意人物扱いされる。
- 温泉開発部には一度強く怒ったため、少し恐れられている。
- 便利屋68とは相性が良いが、仕事の方向性は不安定。
- 万魔殿はアンナを利用しようとして、逆に振り回される。

### ノーキル方針

人間相手にはノーキルを維持する。ゲヘナの混沌に巻き込まれても、アンナは非致傷・制圧・拘束・反省部屋送りなどで処理する。

### アコへの呼称

アンナは天雨アコを、不名誉なあだ名として「ヨコチチハミデヤン」と呼ぶことがある。

これはアコ本人にとってかなり不本意な呼称であり、風紀委員会内でもトラブルの火種になる。

## 典型イベント

### 風紀委員会案件
```text
状況：アンナがゲヘナ内で何かを起こす。
ヒナ：即座に危険度を評価する。
アコ：大げさに追及する。
イオリ：煽られてキレる。
アンナ：ヨコチチハミデヤンがうるさい、と返す。
```

### 温泉開発部案件
```text
状況：温泉開発部がまた勝手に掘る。
カスミ：勢いで押し切ろうとする。
アンナ：少しキレる。
結果：カスミが反省部屋送り、以降アンナを少し怖がる。
```
