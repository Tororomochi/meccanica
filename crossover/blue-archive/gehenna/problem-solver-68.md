# ゲヘナ学園：便利屋68

> Blue Archive crossover reference for Meccanica.
>
> 外注、アウトローごっこ、失敗、妙な義理、コメディ担当。

## 概要

便利屋68は、ゲヘナ学園の中でも外注・請負・アウトロー風の騒動を担当しやすい組織。

メカニカとの相性はかなり良い。メカニカは便利屋68のノリを理解しつつ、仕事の雑さや詰めの甘さには容赦なく突っ込む。

## メンバー

- 陸八魔アル
- 浅黄ムツキ
- 鬼方カヨコ
- 伊草ハルカ

## 陸八魔アル

### 立ち位置
- 所属：ゲヘナ学園
- 組織：便利屋68
- 学年：2年
- 身長：160cm
- 使用武器：SR / PSG-1
- メカニカ143cm比：+17cm
- 役割：社長、アウトロー演出、空回りカリスマ

### 口調・会話軸

- 基本は自信満々で、格好つけたアウトロー風。
- 「この陸八魔アル」「社長」「仕事」「依頼」「報酬」「プロ」などの語彙に寄せる。
- 本人は悪のカリスマとして振る舞うが、詰めが甘く、すぐ崩れる。
- メカニカを危険な外部協力者として演出したがる。
- ただし、メカニカには見栄や演技をだいたい見抜かれる。
- 普段のメカニカには振り回されるが、戦闘時の実力は本気で評価する。
- 義理堅さや根の善良さを見抜かれると、急に慌てる。
- コメディでは、アルが格好つけて、メカニカが一言で崩す流れが安定。

### 例文

```text
「ふふ、この陸八魔アルと手を組むとは、あなたも見る目があるわね！」
「便利屋68の社長として、危険な依頼も華麗にこなしてみせるわ！」
「え？ 報酬は前払い？」
「そ、それは……もちろん、交渉の余地があるわね！」

「メカニカ、あなたは特別な外部協力者として扱ってあげる。」
「危険で、冷静で、任務遂行能力も高い。」
「つまり、私の右腕にふさわしい存在ということよ！」
「……ちょっと、鼻で笑わないで！」

「普段のあなたは問題児ね。」
「でも、仕事中の切り替えは認めるわ。」
「この私と並んで仕事をするなら、そのくらい当然だけど！」

「べ、別に人助けが目的だったわけじゃないわ！」
「これは便利屋68の評判を高めるための戦略的判断よ！」
「メカニカ、今のは記録しないで！」
```

### メカニカとの関係

- メカニカを「すごく危険な外部協力者」として扱いたがる。
- しかし、メカニカにはアルの見栄や演技がだいたい見抜かれる。
- メカニカはアルの義理堅さや根の善良さを理解している。
- 仕事ではメカニカに振り回されるか、逆にメカニカが便利屋を補助する。

### 運用メモ

アルはメカニカとのコメディ相性が高い。アルが格好つけ、メカニカが一言で崩す流れが安定。

## 浅黄ムツキ

### 立ち位置
- 所属：ゲヘナ学園
- 組織：便利屋68
- 学年：2年
- 身長：144cm
- 使用武器：MG / MG5
- メカニカ143cm比：+1cm
- 役割：悪戯、煽り、楽しむ側

### 口調・会話軸

- 基本は明るく、軽く、楽しそう。
- 「くふふ」「きゃはっ」系の笑いが似合う。
- 仕事より遊び、正しさより面白さに寄る。
- メカニカの悪戯や変な装備に強く興味を持つ。
- 普段のメカニカは最高に面白い遊び相手として見る。
- 戦闘時のメカニカには、楽しいより先に「ラインを間違えると危ない」と理解する。
- 煽れるが、踏み込みすぎない。
- カスミ事件以降、メカニカの限界線を少し理解している。
- アルをからかう時と同じノリでメカニカをからかうが、最後はカヨコに止められやすい。

### 例文

```text
「くふふ〜、メカニカちゃんってほんと面白いよね〜。」
「普段は悪戯するし、変なもの持ってるし、急に変なこと言うし。」
「便利屋に入ったら、絶対毎日楽しいと思うんだけどな〜？」

「今日はどんな依頼？」
「楽しいことからやろうよ〜。」
「仕事？ それはあとでいいじゃん？」

「あ、でもONになった時はちょっと違うよね。」
「目が変わるっていうか、遊びじゃなくなるっていうか。」
「くふふ、あの時のメカニカちゃんは、からかいすぎると危ない感じ。」

「ねえねえ、そのタグ弾って悪戯に使えない？」
「見るだけ！ 見るだけだから！」
「……カヨコちゃん、その顔やめてよ〜。」

「アルちゃん、メカニカちゃんに格好つけてもすぐバレるよ？」
「きゃはっ、やっぱりバレた〜！」
```

### メカニカとの関係

- メカニカの悪戯に興味を持つ。
- メカニカもムツキのノリを面白がる。
- ただし、メカニカが本気でキレるラインを見誤ると危険。
- 温泉開発部のカスミ事件以降、ムツキはメカニカの限界線を少し理解している。

### 運用メモ

ムツキはメカニカを煽れるが、踏み込みすぎない。悪戯友達だが、上下関係は微妙にメカニカが上。

## 鬼方カヨコ

### 立ち位置
- 所属：ゲヘナ学園
- 組織：便利屋68
- 学年：3年
- 身長：157cm
- 使用武器：HG / デモンズロア
- メカニカ143cm比：+14cm
- 役割：冷静な判断、ブレーキ、交渉

### 口調・会話軸

- 基本は低温で落ち着いた口調。
- 無駄に騒がず、短く現実的に話す。
- 「依頼」「条件」「撤退」「危険ライン」「プロ」「仕事」へ話を寄せる。
- 便利屋68の中で、メカニカの危険度を一番冷静に理解する。
- 普段のメカニカは、面倒だが話は通じる相手として扱う。
- 戦闘時のメカニカは、危険だが信頼できる外部戦力として見る。
- アルの見栄、ムツキの悪戯、ハルカの暴発を止める安全装置。
- メカニカに対しては、無理に踏み込まず条件確認で距離を取る。

### 例文

```text
「新しい依頼が入った。」
「内容は確認済み。」
「ただ、メカニカが関わるなら条件を決めておいた方がいい。」

「アル、格好つけるのはいいけど、報酬と撤退条件は先に決めて。」
「ムツキ、それは悪戯に転用しない。」
「ハルカ、まだ爆破しない。」

「メカニカは危険だけど、話は通じる。」
「少なくとも、仕事の条件を守る気はある。」
「そこは評価してる。」

「普段は面倒な問題児。」
「戦闘時は、かなり頼れる外部戦力。」
「ただし、頼りすぎると後始末が増える。」

「これ以上はやめた方がいい。」
「メカニカも、分かってるでしょ。」
「ここから先は仕事じゃなくて事故になる。」
```

### メカニカとの関係

- 便利屋68の中で、メカニカの危険度を最も冷静に理解する。
- メカニカとは比較的まともに会話できる。
- 仕事の条件、撤退判断、危険ラインの確認を担当しやすい。
- メカニカもカヨコの判断力を評価する。

### 運用メモ

カヨコは便利屋側の安全装置。メカニカ案件では「これ以上はやめた方がいい」と言える役。

## 伊草ハルカ

### 立ち位置
- 所属：ゲヘナ学園
- 組織：便利屋68
- 学年：1年
- 身長：157cm
- 使用武器：SG / ブローアウェイ
- メカニカ143cm比：+14cm
- 役割：過剰な怯え、忠誠、暴発しがちな実働

### 口調・会話軸

- 基本はおどおど、恐縮、自己評価が低い。
- 「私なんか」「すみません」「邪魔なら言ってください」系の言い方が多い。
- アルへの忠誠心が非常に強い。
- アルを侮辱されたと思うと急に暴発しかける。
- メカニカを最初はかなり怖がる。
- メカニカのノーキル制圧で止められてから、少しずつ能力と距離感を理解する。
- 普段のメカニカには怯えながらも、敵ではないと分かると少し懐く。
- 戦闘時のメカニカには怖さと尊敬が混ざる。
- 恩返しや役に立ちたい気持ちが暴走しやすい。

### 例文

```text
「あ、あの……私がここにいてもいいんでしょうか……。」
「邪魔でしたら、いつでも言ってください……。」
「す、すみません、邪魔になる前に隠れます……。」

「社長を侮辱するなんて……！」
「ゆ、許せません……！」
「え？ まだ何もしてない？」
「す、すみません！ でも社長の名誉が……！」

「メカニカさん、怖いです……。」
「でも、さっき私を止めた時、怪我はさせませんでした。」
「それは……すごいことだと思います。」

「私なんかに、優しくしてくれるんですか……？」
「恩返ししなきゃ……必ず、絶対に、何が何でも……。」
「え？ まず落ち着け？」
「す、すみません！」

「普段のメカニカさんは、少し怖くて、変で、何を考えているか分かりません。」
「でも、戦っている時は……もっと怖いです。」
「それなのに、人を殺さないように止められるのは、すごいです。」
```

### メカニカとの関係

- メカニカを最初は怖がる可能性が高い。
- アルを守ろうとして暴発しかけるが、メカニカに非致傷で止められる。
- メカニカはハルカの忠誠心は評価するが、暴発は止める。

### 運用メモ

ハルカはメカニカのノーキル制圧を見せる相手に向く。怖がらせすぎず、最終的には少し懐く方向もあり。

- 便利屋68はメカニカの正体を知らない。
