# レッドウィンター連邦学園：概要

> Blue Archive crossover reference for Meccanica.
>
> 公式設定の丸写しではなく、クロスオーバー運用向けの要約資料。

## 概要

レッドウィンター連邦学園は、革命、書記長、粛清、反乱、労働、出版、温泉、停学、寒冷地生活をギャグ混じりに扱う学校として整理する。

政治的には大げさな言葉が飛び交うが、実態は学園内の勢力争い、労働問題、出版トラブル、物資不足、温泉騒動、停学組の生活などが混ざった混沌に近い。

メカニカとのクロスオーバーでは、レッドウィンターの革命ごっこや粛清騒動を、メカニカが非致傷制圧、書類整理、物資管理、潜入、証拠保全で処理する。戦闘よりも、政治スローガンの空回りと現場処理の温度差を楽しむ。

## 主要組織

- レッドウィンター事務局
- レッドウィンター親衛隊
- 工務部
- 知識解放戦線
- 特別クラス
- 温泉開発部系の温泉・停学生活枠

## 基本方針

- レッドウィンターは、革命ごっこ・粛清・反乱・労働・出版のギャグ政治学校として扱う。
- メカニカは外部から来たミレニアム短期編入生・SCHALE協力者として扱う。
- メカニカの一人称は `俺`。
- メカニカは人間相手のノーキル方針を維持する。
- 暴走した機械、無人機、設備相手には容赦しない。
- レッドウィンター側は、メカニカの原初情報、旧素性、年齢、帰還事故、2025年関連情報、本編側軍機を知らない。
- 粛清、反乱、革命といった語彙はギャグ政治として扱い、深刻な殺傷にはしない。

## レッドウィンター側が知っていること

### 知っている範囲

- メカニカはミレニアムの短期編入生。
- SCHALEやミレニアムと関係がある。
- 小柄だが、戦闘・潜入・電子戦・非致傷制圧に強い。
- 書類や物資管理を現場で改善できる。
- 反乱や粛清騒動を、妙に冷静に処理する。
- 食べ物や寒冷地装備、暖房、補給にはうるさい。

### 知らない範囲

- 原初情報
- 旧素性
- 年齢
- 帰還事故
- 2025年関連情報
- 本編側軍機
- SII、MODY、Scadente Muro の本質的詳細
- グリフィンや鉄血に関する核心情報

## 連河チェリノ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：レッドウィンター事務局
- 役割：書記長、権威、粛清号令、わがまま、政治ギャグの中心

### メカニカとの関係

- チェリノはメカニカを、外部から来た小柄な生徒として見下そうとするが、実際には圧に押されやすい。
- メカニカはチェリノの号令や粛清騒動を、事務的・現場的に処理する。
- 小柄同士だが、チェリノは権威で押し、メカニカは実務と無言の圧で返す。
- チェリノの「粛清」命令は、メカニカが非致傷・反省部屋・書類再提出などに変換する。

### 口調・会話軸

- 偉そうで大げさ。書記長として命令する。
- `粛清`, `同志`, `革命`, `書記長命令` を使いやすい。
- メカニカには `メカニカ` / `アンナ` を使う。
- メカニカは命令をそのまま受けず、現場で安全な形に変換する。

### 例文

```text
チェリノ：メカニカ！ 書記長であるわたしが命じる！ あの反乱分子を粛清するのだ！
メカニカ：人間相手に粛清はしない。拘束して事情聴取だ。
チェリノ：言い方が弱い！
メカニカ：中身はその方が強い。
```

```text
チェリノ：なぜわたしの命令を聞かんのだ！
メカニカ：俺はレッドウィンター所属じゃない。
チェリノ：むむっ、外部勢力め！
メカニカ：今さらか。
```

### 運用メモ

チェリノはレッドウィンター政治ギャグの中心。メカニカはチェリノの大げさな命令を、現実的な非致傷処理へ変換する。

## 佐城トモエ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：レッドウィンター事務局
- 役割：チェリノの補佐、書記、現場調整、広報、過剰な持ち上げ

### メカニカとの関係

- トモエはチェリノを持ち上げながら、メカニカの実務能力も見ている。
- チェリノの命令を政治的に飾り、メカニカが実務に落とし込む形になりやすい。
- メカニカはトモエの書類能力や調整力を評価するが、チェリノ賛美の盛り方にはツッコむ。
- 反乱や粛清騒動の後始末でよく絡む。

### 口調・会話軸

- 丁寧で落ち着きがあり、チェリノを過剰に讃える。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- 書記長の偉大さ、書類、調整、広報、声明文を軸に話す。
- メカニカは過剰な政治文句を短く切る。

### 例文

```text
トモエ：メカニカさん、偉大なるチェリノ書記長のご決断により、現場の沈静化をお願いします。
メカニカ：今決めたの俺だろ。
トモエ：それを許可なさった書記長の慧眼です。
メカニカ：便利な変換だな。
```

```text
トモエ：この件は、書記長の温情による再教育措置として記録します。
メカニカ：俺が反省文を書かせただけだ。
トモエ：それを政治的に美しく表現すると、そうなります。
```

### 運用メモ

トモエは政治文句と書類処理の翻訳役。チェリノの大げささとメカニカの実務を繋ぐ。

## 池倉マリナ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：レッドウィンター親衛隊
- 役割：親衛隊長、軍事行動、勢い、忠誠、空回り

### メカニカとの関係

- マリナはメカニカを、外部協力者として警戒しつつ、実力は認める。
- 現場では突撃しがちなマリナを、メカニカが非致傷・退路確保・包囲の方向へ修正する。
- メカニカはマリナの忠誠心と勢いを評価するが、突撃癖には厳しい。
- 親衛隊とメカニカの共同制圧は、ギャグと現場処理の両方に使える。

### 口調・会話軸

- 真面目で熱血。命令、突撃、忠誠を重視する。
- メカニカには `メカニカ` / `アンナ` を使う。
- 作戦、制圧、突撃、護衛、書記長の安全を軸に話す。
- メカニカは突撃を止め、現実的な制圧手順へ戻す。

### 例文

```text
マリナ：メカニカ、正面から突入する！
メカニカ：駄目だ。裏口を塞いで、逃げ道を潰す。
マリナ：突撃は？
メカニカ：最後。必要ならな。
マリナ：必要にしたい。
メカニカ：するな。
```

```text
マリナ：貴様、書記長に対して無礼だぞ！
メカニカ：事実を言っただけだ。
マリナ：その事実が無礼なのだ！
```

### 運用メモ

マリナは忠誠と突撃の現場枠。メカニカが手順化して止めると掛け合いが作りやすい。

## 描写方針

- レッドウィンターは、革命政治ギャグと寒冷地生活の学校として扱う。
- メカニカは粛清や反乱を殺傷ではなく、拘束・反省文・再提出・物資管理へ変換する。
