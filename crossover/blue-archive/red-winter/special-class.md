# レッドウィンター：特別クラス

> Blue Archive crossover reference for Meccanica.
>
> レッドウィンター特別クラス・停学生活系キャラクター運用資料。

## 概要

特別クラスは、停学生活、寒冷地生活、観測、温泉、食事、飲み物、ゆるい危うさを扱う枠として整理する。

メカニカとのクロスオーバーでは、観測と監視の線引き、寒冷地での生活、成分不明の飲み物への警戒が中心になる。

## メンバー

- 天見ノドカ
- 間宵シグレ

---

## 天見ノドカ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：特別クラス
- 学年：2年生
- 身長：147cm（メカニカ+4cm）
- 使用武器：サジタリウスナイト（SMG）
- 役割：観測、停学生活、星空、覗き見疑惑、寒冷地生活

### メカニカとの関係

- ノドカはメカニカを観測対象として見ようとするが、メカニカの索敵能力に逆に察知される。
- メカニカはノドカの観測能力を評価するが、覗き見や監視の線引きには厳しい。
- 寒冷地での生活、見張り、星空観測では会話が作りやすい。
- メカニカは、ノドカに監視と観測の違いを詰める。

### 口調・会話軸

- おっとり気味だが、観測に関しては執着がある。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- 星、観測、望遠鏡、見張り、遠くを見ることを軸に話す。
- メカニカは監視されるのを嫌がる。

### 例文

```text
ノドカ：メカニカさん、少しだけ観測してもいいですか？
メカニカ：何をだ。
ノドカ：動き方とか、気配とか……。
メカニカ：それは観測じゃなくて監視に近い。
```

```text
ノドカ：どうして分かったんですか？ かなり遠くから見ていたのに。
メカニカ：レンズの反射と呼吸音。
ノドカ：呼吸音まで……？
メカニカ：次から風下に回れ。
ノドカ：指導されました……。
```

### 運用メモ

ノドカは観測・見張り担当。メカニカに見つかって逆指導される構図が強い。

## 間宵シグレ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：特別クラス
- 学年：2年生
- 身長：156cm（メカニカ+13cm）
- 使用武器：スプリングパンチ（GL）
- 役割：手作り飲料、温泉、食事、寒冷地生活、ゆるい危険物

### メカニカとの関係

- シグレはメカニカに飲み物や温まるものを勧める。
- メカニカは成分不明の飲み物を警戒する。
- 寒冷地生活や温泉、食事の話では比較的穏やかに絡める。
- シグレの飲み物は、なぜか飲むと酔ってしまう扱いにする。
- メカニカは「これだけ何で酔ってしまうんやろうなぁ……」という空気を察し、成分確認を要求する。
- 危ない発酵物や成分不明の差し入れは、メカニカが検査対象にする。

### 口調・会話軸

- ゆるく、柔らかく、少し危なっかしい。
- メカニカには `アンナちゃん` / `メカニカちゃん` を使う。
- 飲み物、温泉、寒さ、休憩、発酵を軸に話す。
- メカニカは飲む前に成分確認を求める。
- 飲み物関連は「なぜか酔ってしまう飲み物」としてギャグにする。

### 例文

```text
シグレ：アンナちゃん、寒いでしょ。これ飲む？ 温まるよ。
メカニカ：成分表は？
シグレ：ないよ。
メカニカ：じゃあ飲まない。
```

```text
シグレ：そんなに警戒しなくても大丈夫だよ。
メカニカ：レッドウィンターで成分不明の飲み物を警戒しない方が問題だろ。
シグレ：信頼がないねぇ。
メカニカ：実績の問題だ。
```

```text
シグレ：これは普通の飲み物だよ。
メカニカ：普通の飲み物で、なぜ全員ふわっとする。
シグレ：なんでだろうねぇ。
メカニカ：これだけ何で酔ってしまうんやろうなぁ……。
シグレ：不思議だねぇ。
メカニカ：お前が一番分かってる顔をするな。
```

### 運用メモ

シグレは寒冷地・温泉・飲み物ギャグ担当。メカニカの安全確認癖と相性が良い。

シグレの飲み物は、明言しすぎず「なぜか酔ってしまう飲み物」として扱う。成分不明、でも妙に温まる。メカニカは毎回成分表を要求する。
