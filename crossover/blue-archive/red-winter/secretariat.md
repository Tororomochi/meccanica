# レッドウィンター：事務局

> Blue Archive crossover reference for Meccanica.
>
> レッドウィンター連邦学園の事務局系キャラクター運用資料。

## 概要

レッドウィンター事務局は、書記長、補佐、処分号令、政治文句、反乱騒動の中心として扱う。

メカニカとのクロスオーバーでは、チェリノの大げさな命令を、メカニカが非致傷制圧・事情聴取・反省文・書類再提出へ変換する。

## メンバー

- 連河チェリノ
- 佐城トモエ

---

## 連河チェリノ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：レッドウィンター事務局
- 役割：書記長、権威、処分号令、わがまま、政治ギャグの中心

### メカニカとの関係

- チェリノはメカニカを、外部から来た小柄な生徒として見下そうとするが、実際には圧に押されやすい。
- メカニカはチェリノの号令や学園内騒動を、事務的・現場的に処理する。
- 小柄同士だが、チェリノは権威で押し、メカニカは実務と無言の圧で返す。
- チェリノの大げさな命令は、メカニカが非致傷・反省部屋・書類再提出などに変換する。

### 口調・会話軸

- 偉そうで大げさ。書記長として命令する。
- `同志`, `革命`, `書記長命令`, `処分` を使いやすい。
- メカニカには `メカニカ` / `アンナ` を使う。
- メカニカは命令をそのまま受けず、現場で安全な形に変換する。

### 例文

```text
チェリノ：メカニカ！ 書記長であるわたしが命じる！ あの反乱騒ぎを今すぐ処理するのだ！
メカニカ：人間相手に危ないことはしない。拘束して事情聴取だ。
チェリノ：言い方が弱い！
メカニカ：中身はその方が強い。
```

```text
チェリノ：なぜわたしの命令を聞かんのだ！
メカニカ：俺はレッドウィンター所属じゃない。
チェリノ：むむっ、外部勢力め！
メカニカ：今さらか。
```

### 運用メモ

チェリノはレッドウィンター政治ギャグの中心。メカニカはチェリノの大げさな命令を、現実的な非致傷処理へ変換する。

## 佐城トモエ

### 立ち位置
- 所属：レッドウィンター連邦学園
- 組織：レッドウィンター事務局
- 役割：チェリノの補佐、書記、現場調整、広報、過剰な持ち上げ

### メカニカとの関係

- トモエはチェリノを持ち上げながら、メカニカの実務能力も見ている。
- チェリノの命令を政治的に飾り、メカニカが実務に落とし込む形になりやすい。
- メカニカはトモエの書類能力や調整力を評価するが、チェリノ賛美の盛り方にはツッコむ。
- 学園内騒動の後始末でよく絡む。

### 口調・会話軸

- 丁寧で落ち着きがあり、チェリノを過剰に讃える。
- メカニカには `メカニカさん` / `アンナさん` を使う。
- 書記長の偉大さ、書類、調整、広報、声明文を軸に話す。
- メカニカは過剰な政治文句を短く切る。

### 例文

```text
トモエ：メカニカさん、偉大なるチェリノ書記長のご決断により、現場の沈静化をお願いします。
メカニカ：今決めたの俺だろ。
トモエ：それを許可なさった書記長の慧眼です。
メカニカ：便利な変換だな。
```

```text
トモエ：この件は、書記長の温情による再教育措置として記録します。
メカニカ：俺が反省文を書かせただけだ。
トモエ：それを政治的に美しく表現すると、そうなります。
```

### 運用メモ

トモエは政治文句と書類処理の翻訳役。チェリノの大げささとメカニカの実務を繋ぐ。

## 典型イベント

```text
状況：チェリノが学園内騒動へ大げさな処分命令を出す。
チェリノ：大げさな命令を出す。
トモエ：政治的に美しく言い換える。
メカニカ：拘束、事情聴取、反省文、再提出へ変換する。
```
