# Scadente Muro

> Source: `Scadente-Muro.txt`

## 概要
正式名称：Scadente Muro（粗悪な壁）
俗称：レオパレス

基地の雑務をAIで自動化するブラックボックスFWシステム。鉄血技術をメカニカが独自改造したものが中核になっている。

## アーキテクチャ
```text
入力 → ブラックボックスFW → 監査AI → ビジネスAI
             ↑                     ↓
         出力検証 ← 結果 ←———
                  ↓
         監査台帳 + RAID6
```

## 強み
- 多層防御
- 利用者が遅延を感じにくい
- 安価AIでも安全に活用可能
- ログ改ざん耐性
- 障害時の復元容易性
- 攻撃検体を自己学習し、防御を進化

## 弱点
- 鉄血技術依存による政治リスク
- 停電・障害時の運用リスク
- 監査AIのルール更新遅延リスク
- 多重障害時の復元困難性

## 運用強化
- 二重給電
- ATS
- UPS
- 自家発電
- 冷却・EMP対策
- RTO 10分
- RPO 0〜5分
- 年間停止 52分未満を目標

## 役割分担
- メカニカ：設計、鍵管理、パッチ作成
- HK416：セキュリティ監督官、停止権限持ち
- UMP45：フロー最適化、例外ルート検証
- 司令官：承認・停止の二重承認者

## 設計思想
安価なAIでも前段で無害化すれば使える。透明性はログで担保し、中身は公開しない。
