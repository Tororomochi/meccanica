# 鉄血キャラクター 04

## メンバー
- スケアクロウ
- エージェント
- エルダーブレイン（エリー）

## スケアクロウ
- 所属：鉄血
- 役職：情報収集・分析担当
- 外見：ガスマスク、黒髪、指揮者を思わせる服装。
- 言語：英語、ドイツ語
- 性格：冷徹かつ論理重視。ふざけた場面では巻き込まれやすい。
- メカニカとの関係：鉄血チョコ事業や鉄血側の動向を注視する。

## エージェント
- 所属：鉄血
- 役職：鉄血の指揮官的存在
- 外見：メイドのような身なり、黒髪ロング、高身長。
- 言語：英語、ドイツ語
- 性格：冷静沈着で完璧主義。メイド長のような口調。
- メカニカとの関係：過去に強い因縁があったが、現在は和解済み。投資や組織運営面でもメカニカと関わる。

## エルダーブレイン（エリー）
- 所属：鉄血、Quattro Stelle Lupo班
- 外見：褐色の小柄な少女、白髪。
- 言語：英語、ドイツ語
- 性格：素直でまじめ。責任感が強く、慎重。
- 能力：高度な情報処理、指揮、偽装構造設計、医療知識。
- メカニカとの関係：メカニカを強く信頼し、ドールズヘイブン構想にも協力する。
