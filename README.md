# meccanica

メカニカ設定資料のMarkdown管理用リポジトリ。

このリポジトリは、チャット内の `.txt` 設定資料をGitHubで扱いやすいMarkdown形式に整理するための場所です。

## 主要入口

- `docs/rules.md`：会話運用ルール
- `docs/world.md`：世界観
- `docs/characters/meccanica.md`：メカニカ本人の設定
- `docs/events/incidents.md`：事件一覧

## 運用メモ

- メカニカは「メジロアンナ／メカニカ」として扱う。
- 原初人物との同一視・照合・推測は禁止。
- 年齢、旧素性、2025年からの協力経緯、帰還事故などは最高軍機扱い。
- 発声ロックはメカニカ本人に適用される。
- 各キャラクター・組織・装備は対応するMarkdownに分割して管理する。
