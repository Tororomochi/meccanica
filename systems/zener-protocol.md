# ツェナープロトコル

> Meccanica system reference.
>
> メカニカ内部通信、携帯回線プラグイン、ケイ互換通信などを扱う基幹通信プロトコル。

## 概要

ツェナープロトコルは、メカニカ内部の基幹通信レイヤーである。

メカニカは物理的なスマートフォンを持たなくても、ツェナープロトコル上で通話、SMS、データ通信、認証済みノードとの通信を処理できる。

ツェナープロトコルそのものは携帯電話網ではない。メカニカ内部の通信OS、認証回線、戦術人形系通信バスに近い。

外部の通常回線へ出る場合は、携帯回線プラグインを経由する。

## 携帯電話としての通信経路

通常の携帯電話として外部へ発信する場合、通信はツェナープロトコル上で処理された後、携帯回線プラグインを経由して基地局へ送られる。

通信経路は以下の通り。

```text
メカニカ本体
→ ツェナープロトコル
→ 携帯回線プラグイン
→ 基地局
→ 相手のスマートフォン
```

外部の相手からは、通常のスマートフォンや携帯電話からの通話と同じように見える。

メカニカ側には物理的なスマートフォンが存在しないため、着信や通知は内部HUD、聴覚系、音声出力、内部ログ上のイベントとして扱われる。

## 携帯回線プラグイン

携帯回線プラグインは、ツェナープロトコルを一般の携帯電話網へ接続する外部変換層である。

スマートフォンでいうSIM/eSIM、モデム、電話アプリ、通信スタックに相当する機能を担う。

ただし、ツェナープロトコル本体とは別扱いであり、外部携帯網への接続はこのプラグインに依存する。

そのため、端末を没収してもメカニカ本体の通信機能は完全には止まらない。一方で、携帯回線プラグインの使用、帯域、接続先、使用時間、外部ゲートウェイへの接続は制限・監査できる。

## できること

ツェナープロトコルと携帯回線プラグインを組み合わせることで、メカニカは以下を実行できる。

- 音声通話
- SMS
- データ通信
- 緊急通信
- 位置情報送信
- 認証済み連絡先との優先通信
- 認証済みノードとのデータ送受信

これらは外部から見ると通常の携帯電話・スマートフォン通信に見えるが、メカニカ内部ではツェナープロトコル上の通信イベントとして処理される。

## 制限と監査

ツェナープロトコルそのものはメカニカ本体内部の通信基盤であるため、外部から完全停止させることは難しい。

一方で、携帯回線プラグイン、学内ネットワーク接続、外部ゲートウェイの使用は制限・監査できる。

監査側が確認できるのは主に以下。

- 通信の発生
- 通信量
- 接続先種別
- 使用プラグイン
- 通信時間
- 外部ネットワークへの接続有無

通信内容は基本的に暗号化されており、通常の監査では読めない。

```text
ツェナープロトコル停止：原則不可／危険
携帯回線プラグイン停止：可能
学内ネットワーク接続停止：可能
通信ログ外形監査：可能
通信内容読取：原則不可
```

## ケイとの互換

現時点で、ツェナープロトコルを使用可能なブルーアーカイブ側存在は天童ケイのみ。

ケイは携帯回線プラグインを経由せず、ツェナープロトコル互換層を通じてメカニカと直接通信できる。

```text
メカニカ本体
→ ツェナープロトコル
→ ケイ
```

アリスにはツェナープロトコル互換は提供されていない。

これは、アリスを戦術人形系通信網や軍用情報系統へ不用意に接続しないための保護措置である。

ケイはアリスを守る立場にあるため、自身が互換層を扱える場合でも、アリスへ同等のアクセス権を渡すことはしない。

## セミナー・ヴェリタスから見た扱い

セミナーやヴェリタスが制限できるのは、主に外部接続部分である。

端末没収、学内Wi-Fi遮断、携帯回線プラグイン停止、外部ゲートウェイ制限は可能だが、メカニカ本体内部のツェナープロトコルを完全停止させることは基本的に想定しない。

そのため、ミレニアム側の運用では以下のような扱いになる。

- 通常回線は制限対象。
- 緊急通信は原則維持。
- 学内ネットワークへの接続は申告制。
- 高帯域通信は監査対象。
- 通信内容ではなく、通信の外形ログを監査する。
- ケイとの直接通信は、必要に応じて特別枠で扱う。

## 描写上の使い方

ツェナープロトコルは、メカニカが「スマートフォンを持っていないのに通話できる」理由になる。

外部から見れば普通の電話だが、メカニカ本人は本体内部の通信イベントとして処理している。

```text
ユウカ「スマホは？」
アンナ「ない」
ユウカ「では今の通話は？」
アンナ「俺」
ユウカ「あなた？」
アンナ「俺がスマホ」
ノア「書類が増えましたね」
```

この設定により、メカニカの身体そのものが軍用通信インフラであること、そしてミレニアム側から見ればセミナー案件であることを描写できる。
